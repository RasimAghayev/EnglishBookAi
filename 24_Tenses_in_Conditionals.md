---

# 1. 📌 Introduction / Giriş

A **conditional sentence** has two parts:

```
   IF + condition  ,  result
   ┌─────────────┐    ┌──────────────┐
   If it rains   ,    I will stay home.
   (the CONDITION)    (the RESULT)
   şərt               nəticə
```

The big question conditionals answer is always:

> **"Under what condition does something happen?"**
> 🇦🇿 *"Hansı şərtlə nə baş verir?"*

English has **four main types** + **mixed** types. They differ in **how real** the situation is —
from a scientific fact (100% true) all the way to an impossible past regret (0% — it never
happened). The **tense** you choose is what signals that "reality level." 🎚️

```
   REAL ────────────────────────────────────────────── UNREAL
   Zero      First        Second         Third
   (always)  (likely)     (imaginary)    (impossible past)
```

---

# 2. 🤔 Why do we use them? / Niyə işlədirik?

Because life is full of **"if"**. We make promises, plans, predictions, give advice, imagine other
lives, and regret the past:

```
   FACT:       If you heat water, it boils.            (zero)
   PLAN:       If I finish early, I'll call you.        (first)
   DREAM:      If I won the lottery, I'd buy a house.   (second)
   REGRET:     If I had studied, I would have passed.   (third)
```

Each one needs a **different tense combination** to send the right message. Use the wrong tense and
the meaning changes — *"If I have money"* (maybe I do) vs *"If I had money"* (I don't). One word
flips reality. 💡

# 3. 🌍 Real-life situations / Həyati vəziyyətlər

| Type | When you use it | Example |
|---|---|---|
| 0️⃣ Zero | facts, science, rules, instructions | If you press this, the light turns on. |
| 1️⃣ First | real future plans, promises, warnings | If it's sunny, we'll go to the beach. |
| 2️⃣ Second | dreams, advice, imaginary present | If I were you, I'd apologize. |
| 3️⃣ Third | past regrets, criticism, "what if" | If I'd known, I would have helped. |
| 🔀 Mixed | past cause → present result (& vice-versa) | If I had saved money, I'd be rich now. |

# 4. ⏳ Timeline / Zaman xətti — the reality scale

```
   0️⃣ ZERO     │ always true         │  ●━━━━━━━━━━━━━━●  (any time, 100%)
   1️⃣ FIRST    │ possible future     │  NOW ──────▶ ◑ (50/50, real)
   2️⃣ SECOND   │ unreal now / unlikely│ NOW ··········◌ (imaginary, ~5%)
   3️⃣ THIRD    │ impossible past      │ ◌·········· PAST (0%, didn't happen)
```

> 🇦🇿 Sağa getdikcə **reallıq azalır**: zero = həmişə doğru → third = heç vaxt olmadı.

# 5. 🧪 Formula / Düstur — the five patterns

### 0️⃣ ZERO conditional — general truth
```
   If + Present Simple , Present Simple
   If you mix red and blue, you get purple.
   🇦🇿  Qırmızı ilə mavini qarışdırsan, bənövşəyi alırsan.
   (you can replace "if" with "when" — same meaning)
```

### 1️⃣ FIRST conditional — real future
```
   If + Present Simple , will + V1
   If it rains tomorrow, I will stay home.
   🇦🇿  Sabah yağış yağsa, evdə qalacam.
   ⚠️ NO "will" in the if-clause:  ❌ If it WILL rain...  ✅ If it rains...
```

### 2️⃣ SECOND conditional — unreal / unlikely present
```
   If + Past Simple , would + V1
   If I had a car, I would drive to work.
   🇦🇿  Maşınım olsaydı, işə maşınla gedərdim.
   ⚠️ Use "were" for all persons:  If I WERE you, I would wait.
```

### 3️⃣ THIRD conditional — unreal past (regret)
```
   If + Past Perfect , would have + V3
   If I had studied, I would have passed the exam.
   🇦🇿  Oxusaydım, imtahandan keçərdim (keçmiş olardım).
   ⚠️ NO "would" in the if-clause:  ❌ If I would have known...  ✅ If I had known...
```

### 🔀 MIXED conditional — times crossed
```
 (a) Past condition → Present result:
     If + Past Perfect , would + V1
     If I had saved money, I would be rich now.
     🇦🇿  Pul yığsaydım, indi varlı olardım.

 (b) Present condition → Past result:
     If + Past Simple , would have + V3
     If I were more careful, I wouldn't have crashed.
     🇦🇿  Daha diqqətli olsaydım, qəza etməzdim.
```

### ❓ Questions & ❌ Negatives (any type)
```
   ❓ What will you do if it rains?
   ❓ Would you travel if you had the money?
   ❌ If you don't hurry, you'll miss the bus.   (negative condition)
   ❌ If I hadn't helped, she would have failed.  (negative past)
```

# 6. 🧱 Sentence Structure / Cümlə quruluşu — the COMMA rule

Conditionals work in **both orders**. The order decides the **comma**:

```
   IF-CLAUSE FIRST  →  use a COMMA
   ┌────────────────────┐ ┌──────────────────┐
   │ If it rains        │,│ I will stay home. │
   └────────────────────┘ └──────────────────┘

   RESULT FIRST  →  NO comma
   ┌──────────────────┐ ┌────────────────────┐
   │ I will stay home │ │ if it rains.        │   ← no comma
   └──────────────────┘ └────────────────────┘
```

> 🧠 **Comma rule:** *"IF first → comma; result first → no comma."*
> 🇦🇿 Əgər "if" əvvəldədirsə → vergül qoy; nəticə əvvəldədirsə → vergül qoyma.

---

# 7. 📐 Grammar Rules / Qrammatika qaydaları

1. **No `will` in the if-clause** of zero/first conditionals.
   ❌ *If it will rain* → ✅ *If it rains.* (The condition uses the **present**.)
2. **No `would` in the if-clause** of second/third conditionals.
   ❌ *If I would have time* → ✅ *If I had time.*
3. **Second conditional uses Past Simple** to mean **the present/unreal** — it's not "really" past.
4. **Third conditional uses Past Perfect** (had + V3) for the unreal past.
5. **`were` for everyone** in the second conditional (the subjunctive): *If I/he/she **were**…*
   (informal speech allows *was*, but *were* is correct/safe).
6. **`could` / `might`** can replace **`would`** to show **ability / possibility**:
   *If I had time, I **could** help.* / *…I **might** come.*
7. **`unless` = `if…not`**: *I won't go **unless** you come.* = *…if you **don't** come.*
8. The if-clause and result clause can **swap** (see §6 comma rule).

```
   QUICK MAP:
   if-clause tense   →  result tense
   Present           →  Present           (zero)
   Present           →  will + V1         (first)
   Past Simple       →  would + V1        (second)
   Past Perfect      →  would have + V3   (third)
```

---

# 8. ✍️ Form Rules & "were / was" / Forma qaydaları

| Type | if-clause | main clause |
|---|---|---|
| 0️⃣ | present simple | present simple |
| 1️⃣ | present simple | **will** + V1 |
| 2️⃣ | past simple (**were**) | **would** + V1 |
| 3️⃣ | past perfect (had + V3) | **would have** + V3 |
| 🔀a | past perfect | **would** + V1 (now) |
| 🔀b | past simple | **would have** + V3 |

> 🧠 **"If I were you"** is a fixed, polite advice phrase. Never *"If I was you"* in writing. 🇦🇿
> "Mən sənin yerində olsaydım..." → *If I were you...*

# 9. 🔔 Signal Words / Siqnal sözləri

```
   if            ·  when (zero only)   ·  unless (= if not)
   as long as    ·  provided (that)    ·  in case
   even if       ·  only if            ·  suppose / supposing
   what if...?   ·  imagine if...
```
> 🇦🇿 *əgər, -sa/-sə, məgər, bir şərtlə ki, ...halda, fərz et ki.*

⚠️ **`when` vs `if`:** *when* = it **will** happen (just a question of time); *if* = it **might**.
> *When I get home, I'll call.* (I will get home) vs *If I get home early, I'll call.* (maybe)

# 10. 🚫 Common Mistakes / Ən çox edilən səhvlər

```
 ❌ If it will rain, I will stay home.
 ✅ If it rains, I will stay home.            (no "will" in if-clause)

 ❌ If I would have money, I would travel.
 ✅ If I had money, I would travel.            (second: if + past simple)

 ❌ If I would have known, I would have helped.
 ✅ If I had known, I would have helped.        (third: if + past perfect)

 ❌ If I was you, I would wait.
 ✅ If I were you, I would wait.                (subjunctive "were")

 ❌ If you will not hurry, you will be late.
 ✅ If you don't hurry, you will be late.        (or: Unless you hurry...)

 ❌ I will help you if I will have time.
 ✅ I will help you if I have time.
```

> 🇦🇿 **Ən böyük səhv:** Azərbaycanca "yağacaqsa" deyə düşünüb İngiliscə *"if it **will** rain"*
> demək. İngiliscə şərt hissəsi **gələcək zaman İŞLƏTMİR** — present işlədir.

# 11. ⚖️ Comparison / Müqayisə — which conditional?

```
 ┌─────────┬──────────────────────┬─────────────────────────────────────────┐
 │ TYPE    │ REALITY              │ EXAMPLE                                  │
 ├─────────┼──────────────────────┼─────────────────────────────────────────┤
 │ Zero    │ always true (fact)   │ If you drop it, it breaks.               │
 │ First   │ real, possible future│ If I save enough, I'll buy a car.        │
 │ Second  │ unreal/unlikely now  │ If I had wings, I'd fly. (I don't)       │
 │ Third   │ impossible past      │ If I'd left earlier, I'd have caught it. │
 └─────────┴──────────────────────┴─────────────────────────────────────────┘
```

### 1st vs 2nd — the key feeling
```
   First:   If I have time, I'll help.    → realistic (maybe I will have time)
   Second:  If I had time, I'd help.      → I DON'T have time (imaginary / polite "no")
```

### 2nd vs 3rd — present dream vs past regret
```
   Second:  If I knew her number, I'd call.        → I don't know it NOW
   Third:   If I had known her number, I'd have called. → I didn't know it THEN (past)
```

> 💡 **Decision shortcut:**
> Is it a **fact**? → Zero. · A **real future** chance? → First. · An **imaginary now**? →
> Second. · A **finished past** you can't change? → Third.

# 12. 🇦🇿 Azerbaijani Translation Tips / Tərcümə məsləhətləri

Azerbaijani has the **conditional mood** built in — you already think this way! 🎉

```
   ENGLISH                          AZERBAIJANI
   Zero/First: If it rains...   →   Yağış yağsa...        (-sa/-sə)
   First result: I'll stay      →   qalacam / qalaram     (-acaq / -aram)
   Second: If I had money...    →   Pulum olsaydı...      (-saydı/-səydi)
   Second result: I would buy   →   alardım               (-ardı/-ərdi)
   Third: If I had studied...   →   Oxusaydım...          (-saydı)
   Third result: I'd have passed→   keçərdim / keçmiş olardım
```

⚠️ **Three traps for Azerbaijani speakers:**

1. **No future in the if-clause.** AZ *"yağacaqsa"* tempts you to write *"if it will rain"* — but
   English uses **present**: *"if it rains."*
2. **2nd and 3rd look similar in AZ** (both can use *-saydı*). In English they're **different
   tenses**: 2nd = *if + past simple*, 3rd = *if + past perfect (had + V3)*. Ask: *now* or *past*?
3. **"were", not "was"** in *If I were you*. AZ *"sənin yerində olsaydım"* → always *were*.

> 🇦🇿 Qızıl qayda: **"-sa/-sə" → present (zero/first); "-saydı/-səydi" → past simple (2nd) və ya
> past perfect (3rd). İndi = 2nd, Keçmiş = 3rd.**

# 13. 🔊 Pronunciation Notes / Tələffüz qeydləri

```
   I'll       → /aɪl/   (I will)        I'd      → /aɪd/  (I would / I had)
   would have → /ˈwʊdəv/  ("would-uv")  ⚠️ NOT "would of"!
   I'd have   → /aɪdəv/   ("I-duv")
   wouldn't   → /ˈwʊdənt/
   if I were  → /ɪf aɪ wɜː/
```

> 💡 **Critical:** *would have* sounds like *"would-uv"* /ˈwʊdəv/ — which is WHY many people
> mis-spell it as ❌ *"would of."* It is **always** *would **have*** (or *would've*). 🎯
> 🇦🇿 *would have* → "vud-əv" kimi səslənir, amma **"would of" YAZMA** — düzgünü *would have / would've*.

Contractions you'll hear:
> *I'd go* (= I would) · *I'd gone* (= I had) · *she'd have helped* (= she would have)

# 23. 🎤 Speaking Tips / Danışıq məsləhətləri
1. Use **contractions**: *I'll, I'd, we'd've* — full forms sound robotic.
2. For polite suggestions, reach for the **second conditional**: *"If I were you, I'd…"* — softer
   than a direct command.
3. Say **"would've"** /ˈwʊdəv/, never "would of."
4. *"What would you do if…?"* is a great conversation-starter — practise fast answers.

# 24. 🖊️ Writing Tips / Yazı məsləhətləri
1. Mind the **comma** (§6): *if* first → comma.
2. In essays, the **second conditional** argues hypotheses (*If governments acted, …*), the
   **third** analyses the past (*Had they acted, …*).
3. Use **inversion** (*Were it…, Had we…, Should you…*) sparingly for a formal, advanced tone.
4. Don't mix types by accident — decide your "reality level" first.

# 25. 🎧 Listening Tips / Dinləmə məsləhətləri
1. **'d is ambiguous**: *I'd go* (would) vs *I'd gone* (had). The **next word** tells you:
   *'d + V1* = would; *'d + V3* = had.
2. *would've / could've / might've* blur in fast speech — listen for the **/əv/** ending.
3. **Reduced if-clauses**: natives drop "if" in inversions (*"Had I known…"* = *If I had known*).

# 26. 📖 Reading Tips / Oxu məsləhətləri
1. Spot the **pattern**: the tense in the if-clause tells you the type at a glance.
2. **Inversions** (*Were it not for…, Had he…*) = formal conditionals with the "if" hidden.
3. *unless* = *if not* — mentally flip it to check the meaning.

# 27. 🗽 Native Speaker Usage / Doğma danışanlar necə işlədir
- Natives **mix orders** freely and use **contractions** constantly (*I'd've helped*).
- The **second conditional** is everywhere in advice and politeness (*If I were you…*).
- *"If I'd known…"* is the standard way to express regret/apology.
- In casual speech you'll hear *"was"* for *"were"* (*If I was rich…*) — common, but write *were*.
- **"would of"** is a famous *spelling* error even natives make — it's always **would have**.

# 28. ❓ FAQ / Tez-tez verilən suallar

**Q1. Can I put `will` after `if`?**
Almost never in zero/first. ❌ *If it will rain* → ✅ *If it rains*. (Rare exception: *If it'll
make you happy* = "if it's a question of willingness" — ignore for now.)

**Q2. `was` or `were` in "If I ___ you"?**
✅ **were** (*If I were you*). *was* is heard casually but *were* is correct.

**Q3. What's the difference between 2nd and 3rd?**
2nd = **unreal NOW** (*If I had money…* — I don't, now). 3rd = **unreal PAST** (*If I had had
money…* — I didn't, then). 👉 §11.

**Q4. Is it "would have" or "would of"?**
Always ✅ **would have** (or *would've*). "would of" is wrong — it just *sounds* like it. 👉 §13.

**Q5. What is a mixed conditional?**
The condition and result are in **different times** — e.g. a past cause with a present result:
*"If I had saved money, I'd be rich now."* 👉 §5🔀.

**Q6. `unless` — what does it mean?**
**if…not.** *"Unless you hurry"* = *"if you don't hurry."*

**Q7. How does this map to Azerbaijani?**
*-sa/-sə* → present (zero/first); *-saydı/-səydi* → 2nd (past simple) or 3rd (past perfect). The
trap: AZ doesn't separate 2nd/3rd as clearly → ask "now or past?" 👉 §12.

```
 │ 0️⃣ ZERO   If + present , present            → facts/always true            │

 │                                                                            │
 │ ⚠️ NO "will" / NO "would" in the if-clause.   "If I WERE you."              │
 │ ⚠️ "would HAVE" (never "would of").    Comma when "if" comes first.         │

```

> 🧠 **One-line memory:** *"The further from reality, the further back the tense goes."*
> Present → real. Past → imaginary. Past perfect → impossible. ⏪

# 📚 MINI STORY — *"The Letter He Never Sent"*
### (all conditionals woven together) ✉️

> 🎯 Read the 🇬🇧 first. **Bold** = conditionals (the type is marked in the key). Then read the
> 🇦🇿 and do the questions.

## 🇬🇧 The Letter He Never Sent

Tom found the letter at the bottom of an old box. It was addressed to Sarah, dated thirty years
ago. He had written it but never sent it.

*"If you are reading this,"* it began, *"then I have finally found my courage."*

He smiled sadly. **If he had sent it, everything would have been different.** They **would have
married. They would have grown old together.**

But he hadn't sent it. He had been afraid.

*"If I were braver,"* he had written, *"I would knock on your door tonight."* Even now, the words
hurt. **If only he had knocked.**

He turned the page. *"If you say yes, I will be the happiest man alive. If you say no, I will
understand."* So simple. So young.

His granddaughter came in. "Grandpa, who's Sarah?"

"Someone I knew a long time ago," he said. **"If you ever love someone, send the letter. Don't
wait."**

"Why didn't you send it?"

Tom looked out of the window. **"If I send it now, it's fifty years too late. But if I don't, I'll
always wonder."**

That afternoon, with a shaking hand, he wrote a new address on the envelope. **If she is still
alive, she'll receive it next week. And if she remembers — well, who knows?**

Some letters take thirty years to find their courage too.

## 🔑 Story key — every conditional, decoded
| 🇬🇧 From the story | Type | Why |
|---|---|---|
| If he had sent it, everything would have been different | 3️⃣ Third | unreal past regret |
| They would have married / grown old together | 3️⃣ Third (result) | impossible past |
| If I were braver, I would knock | 2️⃣ Second | unreal present (he isn't brave) |
| If only he had knocked | 3️⃣ Third (wish) | past regret |
| If you say yes, I will be the happiest man | 1️⃣ First | real future possibility |
| If you ever love someone, send the letter | 0️⃣/imperative | general advice/rule |
| If I send it now, it's fifty years too late | 1️⃣ First | real condition + result |
| If she is still alive, she'll receive it | 1️⃣ First | real future |

> 🧠 One story moves through **all four realities** — from impossible past (*if he had sent it*) to
> real future hope (*if she is still alive…*). That's the emotional power of conditionals: they let
> us live the lives we **didn't** choose, and imagine the ones we still might. 💌

# 🧠 MEMORY TRICKS & MNEMONICS
```
 🧠 1. "The further from reality, the further back the tense."
        real future = present (1st) → imaginary now = past (2nd) → impossible = past perfect (3rd)

 🧠 2. "No WILL after IF.  No WOULD after IF."
        if-clause never takes will/would. Condition uses present or past, not "will/would."

 🧠 3. "IF first → comma."   (If it rains, ... )   Result first → no comma.

 🧠 4. "would HAVE, never would OF."   It just sounds like "of."

 🧠 5. "WERE for everyone."   If I/he/she WERE you...

 🧠 6. 🇦🇿  "-sa/-sə = present (0/1) ;  -saydı/-səydi = past (2/3)."
        NOW imaginary → 2nd ;  PAST impossible → 3rd.
```

# 🧩 EXPANSION PACK — *Conditionals* (E1–E12)
### The edges, the elegance, the exam-winners. ☕

> 🎯 §1–§29 gave you the four types + mixed. This pack adds the **advanced moves** that turn a B1
> sentence into a B2/C1 one — and the traps that catch Azerbaijani speakers. Read slowly.

---

## E1 · Inversion — dropping "if" (formal & elegant) 🎩
In formal English you can **delete "if"** and **invert** the subject and auxiliary. Same meaning,
higher register.

```
   2nd:  If I were rich…        →  Were I rich, I would travel.
   3rd:  If I had known…        →  Had I known, I would have come.
   1st:  If you should need…    →  Should you need help, call me.
```
🔑 Only works with **were / had / should**. Great for essays, emails, speeches.
> 🇦🇿 Rəsmi dildə "if" atılır, köməkçi fel önə keçir: *Had I known…* = *If I had known…*

---

## E2 · `could` / `might` / `may` — not everything is "will/would" 🎚️
The result clause can swap **will/would** for a modal to change **certainty or ability**:

```
   If it rains, I might stay home.        (possible, not certain)   ← 1st
   If you study, you could pass.           (ability/possibility)
   If I had a car, I could drive you.      (ability)                ← 2nd
   If I'd left earlier, I might have caught it. (possibility)        ← 3rd
```
```
   will/would  →  100% result
   can/could   →  ability ("would be able to")
   may/might   →  ~50% possibility
```
> 🇦🇿 *will → mütləq; could → bacarardı; might → ola bilər.* Nəticəni "yumşaltmaq" üçün işlət.

---

## E3 · `unless` ≠ always "if not" ⚠️
*unless* usually = *if…not*, **but** it means **"except if."** It doesn't work for **emotional /
unreal** conditions.

```
 ✅ I won't go unless you come.   = if you don't come.        (real condition — fine)
 ❌ I'd be sad unless you came.    ← wrong
 ✅ I'd be sad if you didn't come.                            (use "if not" here)
```
🔑 Use **unless** for **real** conditions about what would *prevent* the result. For feelings /
hypotheticals, stick to **if…not**.
> 🇦🇿 *unless* = "...madıqca / olmasa". Amma hissi/xəyali cümlələrdə **if not** işlət.

---

## E4 · The other "if" words 🔗
```
   as long as / so long as   →  only on this condition  (Stay as long as you're quiet.)
   provided (that) / providing →  on the strict condition (…provided you pay in advance.)
   on condition that          →  formal version          (…on condition that he apologizes.)
   in case                    →  as a precaution         (Take an umbrella in case it rains.)
   even if                    →  it won't matter         (Even if you beg, I won't.)
   only if                    →  the single condition     (Only if you finish will I help.)
```
⚠️ **`in case` ≠ `if`!** *Take an umbrella **in case** it rains* = take it **now**, as a
precaution. *Take an umbrella **if** it rains* = only take it when it's already raining.
> 🇦🇿 *in case* = "hər ehtimala qarşı" (əvvəlcədən); *if* = "əgər (olarsa)".

---

## E5 · `If I were to…` — the formal hypothetical future 🔮
A polished way to introduce an imaginary future proposal (common in IELTS/business):

```
   If I were to accept the offer, I would need a higher salary.
   If the company were to expand, it would create 200 jobs.
```
= *"Imagine, hypothetically, that this happens in the future…"* — softer and more formal than a
plain second conditional. 🇦🇿 *Fərz edək ki, gələcəkdə… olsa…*

---

## E6 · Conditionals can use CONTINUOUS & PERFECT too 🌀
The if-clause and result aren't locked to *simple* tenses — match the **aspect** to the meaning:

```
   If you're looking for Tom, he's in the garden.      (present continuous condition)
   If you've finished, you can go.                       (present perfect condition)
   If it's still raining at six, we'll cancel.            (continuous, 1st)
   If I were working there now, I'd quit.                 (2nd, continuous)
   If I'd been driving faster, I would have crashed.      (3rd, past perfect continuous)
```
> 🧠 The "rules" in §5 are the **skeleton**; you can add **-ing / have + V3** for ongoing or
> completed nuance. The reality-level logic stays the same.

---

## E7 · Conditionals' cousins — `wish` / `if only` 💭
*wish* and *if only* use the **same tense logic** as 2nd/3rd conditionals:

```
   PRESENT regret (like 2nd):  I wish I had a car.        (= but I don't)
                               If only I were taller.
   PAST regret (like 3rd):     I wish I had studied.       (= but I didn't)
                               If only I hadn't said that.
   ANNOYANCE (would):          I wish you would stop that. (complaint about behaviour)
```
🔑 *wish + past simple* = unreal present · *wish + past perfect* = unreal past. Exactly the
backshift of conditionals. (Full treatment isn't here, but the link is worth seeing now.)
> 🇦🇿 *I wish I had…* = "Kaş … olaydı." Same "-saydı" feeling as 2nd/3rd.

---

## E8 · No "if" needed — imperatives & `and` / `or` 🔁
Everyday English hides conditionals inside commands and linkers:

```
   Imperative + and  →  POSITIVE condition:
      Study hard, AND you'll pass.        = If you study hard, you'll pass.

   Imperative + or   →  NEGATIVE condition:
      Hurry up, OR you'll miss the bus.    = If you don't hurry, you'll miss it.

   Bare imperative:
      Touch it and it breaks.              = If you touch it, it breaks.  (zero)
```
> 🇦🇿 *and* = müsbət şərt ("...sən, və..."), *or* = mənfi şərt ("...məsən, yoxsa..."). Çox təbii
> səslənir.

---

## E9 · `should` / `happen to` — for an unlikely real future 🎲
To say a first-conditional event is **unlikely but possible**, add **should** or **happen to** in
the if-clause:

```
   If you should see Tom, tell him to call me.
   If you happen to pass a bakery, buy some bread.
   Should anything go wrong, here's my number.   (inverted — see E1)
```
= *"It probably won't, but just in case…"* — politer and more tentative than plain *if*.
> 🇦🇿 "Birdən… olsa", "təsadüfən… görsən" mənası verir.

## E10 · 🗺️ THE MASTER DECISION MAP — which conditional?
```
 ┌────────────────────────────────────────────────────────────────────────────┐
 │  START: I want to say "if X, then Y."                                       │
 └────────────────────────────────────────────────────────────────────────────┘
        │
        ▼
   Is X always true (a fact, a rule, science)?
        ├── YES → 0️⃣ ZERO    if + present , present        "If you heat ice, it melts."
        └── NO ↓
   Is X a REAL future possibility (it might really happen)?
        ├── YES → 1️⃣ FIRST   if + present , will + V1       "If it rains, I'll stay in."
        └── NO ↓  (it's imaginary / unreal)
   Is the imaginary situation about NOW / the future?
        ├── YES → 2️⃣ SECOND  if + past , would + V1         "If I had money, I'd travel."
        └── NO ↓  (it's about the PAST)
   Is it a finished PAST you can't change?
        ├── YES → 3️⃣ THIRD   if + past perfect , would have+V3  "If I'd known, I'd have come."
        └── Times crossed? → 🔀 MIXED
                 ├── past cause → present result:  if + past perfect , would + V1
                 │       "If I'd saved, I'd be rich now."
                 └── present cause → past result:  if + past , would have + V3
                         "If I were careful, I wouldn't have crashed."
```
🇦🇿 **Açar sual:** *Reallıqmı, xəyalmı? İndimi, keçmişmi?* → tip seçilir.

## E11 · 🏋️ Advanced exercises (inversion, modals, mixed)
```
 1. Rewrite with inversion: "If I had known, I would have called."  → ______
 2. Rewrite with inversion: "If you should need help, ask."         → ______
 3. Soften the result with a modal: "If I had time, I would help."  → (could) ______
 4. Rewrite without "if" (imperative + or): "If you don't leave now, you'll be late." → ______
 5. unless or if not? "I'll be disappointed ______ you don't come." (feelings!)
 6. Mixed: "I didn't learn to swim. I can't join you at the beach now." → If I ______ , I ______ .
 7. Formal future: "Imagine the company expands — 200 jobs would be created."
    → If the company ______ , it ______ .
 8. Add nuance (continuous): "If you (look) ______ for Sara, she's upstairs."
```

<details>
<summary>👉 E11 — Answers & why</summary>

```
 1. Had I known, I would have called.            (had + invert, drop "if")
 2. Should you need help, ask.                    (should + invert)
 3. If I had time, I could help.                  (could = ability)
 4. Leave now, or you'll be late.                 (imperative + or = negative condition)
 5. if not → "...if you don't come."  (NOT "unless" — it's a feeling/unreal result) → E3
 6. If I had learned to swim, I could join you at the beach now.   (mixed: past → present)
 7. If the company were to expand, it would create 200 jobs.       (formal hypothetical, E5)
 8. If you're looking for Sara, she's upstairs.   (present continuous condition, E6)
```
</details>

## E12 · 🇦🇿 Azerbaijani-interference mistakes (deep cut)
```
 ❌ If it will rain tomorrow, I will not go.
 ✅ If it rains tomorrow, I won't go.
    → AZ "yağacaqsa" → "if it will" kimi düşünmə. Şərt = present.

 ❌ If I would be you, I would wait.
 ✅ If I were you, I would wait.
    → "olsaydım" → if-clause-də "would" YOX; "were" işlət.

 ❌ If I would have studied, I would have passed.
 ✅ If I had studied, I would have passed.
    → 3rd-də if-clause = had + V3, "would" deyil.

 ❌ If I had money now, I would have bought it yesterday.  (mismatch)
 ✅ If I had money now, I would buy it.  (2nd)  /  If I had had money then, I would have bought it. (3rd)
    → İndi = 2nd; keçmiş = 3rd. Qarışdırma.

 ❌ When I will arrive, I will call you.
 ✅ When I arrive, I will call you.
    → "when/if" + future YOX (present işlət).
```
> 🧠 The 5 deep AZ traps: **(1)** no future after if/when; **(2)** no *would* in 2nd if-clause —
> use *were*; **(3)** no *would* in 3rd if-clause — use *had + V3*; **(4)** keep 2nd (now) and 3rd
> (past) separate; **(5)** "would have," never "would of."

# 📖 MINI STORY #2 — *"Two Roads"*
### (conditionals as a tool for DECIDING) 🛤️

> 🎯 Story #1 was about *regret* (mostly 3rd). This one is about a **decision** — so it's full of
> **1st** (real plans) and **2nd** (weighing options). **Bold** = conditionals; the type is in the
> key. Read 🇬🇧 first, then 🇦🇿, then the questions.

## 🇬🇧 Two Roads

The email had been open on Nina's screen for an hour. A job offer. In Berlin.

She read it again. **If she accepted, she would leave everything** — her flat, her friends, her
quiet little life in Baku.

"You're overthinking it," said her sister, Aysel.

"It's not that simple. **If I take it, I'll have to move in three weeks.** And **if I move, I won't
see you for months.**"

"And **if you don't take it?** You'll sit here wondering forever."

Nina stared at the ceiling. **If I were braver, I'd have said yes already**, she thought. But she
wasn't sure she was brave.

"Look," Aysel said gently, "**if I were you, I'd go.** You're twenty-six. **If you stay just
because it's safe, you'll regret it.**"

"And what if I fail? What if Berlin is cold and lonely and I can't do the job?"

"Then you come home. **If it doesn't work out, we'll still be here.** Nobody's deleting your room."

Nina laughed for the first time all day.

"**If I'd had this chance ten years ago, I would have run to the airport,**" Aysel added quietly.
"Don't be me. Don't wait until it's a 'would have.'"

That night, Nina didn't sleep. She kept hearing two sentences in her head:

*If I go, I might fail.*
*If I don't go, I'll never know.*

At 6 a.m., as the sky turned grey-gold over the city, she opened the laptop. Her hands were shaking.

She typed three words: *"I accept. Thank you."*

**If you never take the road, you never find out where it goes.**

"O qədər də sadə deyil. Qəbul etsəm, üç həftəyə köçməli olacam. Köçsəm, səni aylarla görməyəcəm."

"Bəs qəbul etməsən? Burada oturub ömrün boyu fikirləşəcəksən."

Nina tavana baxdı. Daha cəsur olsaydım, çoxdan razılaşmışdım, deyə düşündü. Amma cəsur olduğundan
əmin deyildi.

"Bax," Aysel mülayim dedi, "sənin yerində olsaydım, gedərdim. İyirmi altı yaşın var. Sırf təhlükəsiz
olduğu üçün qalsan, peşman olacaqsan."

"Bəs uğursuz olsam? Bəs Berlin soyuq və tənha olsa, işin öhdəsindən gələ bilməsəm?"

"Onda evə qayıdırsan. Alınmasa, biz yenə də burada olacağıq. Heç kim sənin otağını silmir."

Nina bütün gün ərzində ilk dəfə güldü.

"Bu şans on il əvvəl olsaydı, hava limanına qaçardım," deyə Aysel sakitcə əlavə etdi. "Mənim kimi
olma. 'Olardı' deyənə qədər gözləmə."

O gecə Nina yatmadı. Başında iki cümlə təkrarlanırdı:

*Getsəm, uğursuz ola bilərəm.*
*Getməsəm, heç vaxt bilməyəcəm.*

Səhər saat 6-da, şəhərin üzərində səma boz-qızılı rəngə dönəndə, noutbuku açdı. Əlləri əsirdi.

Üç söz yazdı: *"Qəbul edirəm. Təşəkkür edirəm."*

Yola heç vaxt çıxmasan, onun hara apardığını heç vaxt bilməzsən.

## 🔑 Story key — every conditional, decoded
| 🇬🇧 From the story | Type | Why |
|---|---|---|
| If she accepted, she would leave everything | 2️⃣ Second | weighing an imagined option |
| If I take it, I'll have to move in three weeks | 1️⃣ First | real future plan |
| If I move, I won't see you for months | 1️⃣ First | real consequence |
| If you don't take it, you'll sit here wondering | 1️⃣ First | real future |
| If I were braver, I'd have said yes | 🔀 Mixed | present truth → past result |
| If I were you, I'd go | 2️⃣ Second | advice |
| If you stay…, you'll regret it | 1️⃣ First | real warning |
| If it doesn't work out, we'll still be here | 1️⃣ First | reassurance |
| If I'd had this chance ten years ago, I would have run | 3️⃣ Third | past regret |
| If you never take the road, you never find out | 0️⃣ Zero | general truth |

> 🧠 Watch the story **travel the reality scale**: Aysel's regret is a **3rd** (*if I'd had this
> chance…*), her advice is a **2nd** (*if I were you*), the real plans are **1st** (*if I take
> it…*), and the moral is a **0** (*if you never take the road…*). Conditionals are how we **think
> through choices** — that's why they matter. 🛤️

## 🏁 End of Chapter 24

You can now build **all five conditionals** and choose the right one by "reality level":
**zero** (facts) → **first** (real future) → **second** (imaginary now) → **third** (impossible
past) → **mixed** (times crossed). You learned the two golden bans (**no will / no would in the
if-clause**), the **were-for-everyone** subjunctive, the **comma rule**, and **"would have" (never
"would of")**. And you saw it maps onto Azerbaijani **-sa/-sə** and **-saydı/-səydi**. 🔀

```

 │ 2  if + past, would + V1          imaginary now   -saydı → -ardı   │
 │ 3  if + past perfect, would have+V3  impossible past  -saydı → -mış olardı│
 └──────────────────────────────────────────────────────────────────┘
```
