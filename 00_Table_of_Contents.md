## 📑 How This Book Is Organized / Kitab Necə Qurulub

The book is divided into **11 Parts (0–10)**. Each Part contains several **Chapters**.
Each Chapter is one **Markdown file** = one complete, self-contained lesson.

The book does **not** stop after the 12 traditional tenses. Parts 4–7 cover every important
tense-related structure native speakers use, a dedicated **Most Confusing Tenses** comparison, and
an **Alternative Expressions** layer. Part 8 is a full **Final Master Review**; Part 9 holds the
reference appendices; Part 10 is the **Final Mastery Test**.


## ✅ Lesson Blueprint (used in every teaching chapter)

Each teaching chapter contains the core **29-section structure** (Introduction → Why → Real-life →
Timeline → Formula → Structure → Rules → Spelling → Signal Words → Common Mistakes → Comparison →
AZ tips → Pronunciation → Real-life/Daily/Business/Programming/Travel/Academic/IELTS/Formal/Informal
examples → Speaking/Writing/Listening/Reading tips → Native usage → FAQ → Summary).

**Plus the rich practice block:**
- 🎯 CEFR Examples (A1–C2 × 5 EN + 5 AZ) · 📖 Vocabulary · 🗣️ Idioms · Phrasal Verbs · Collocations
- 📚 Mini Story + full AZ translation · 💬 3 Dialogues · ✂️ Chunking · 🎧 Shadowing · 🔊 Pronunciation
- 🧠 Memory Tricks · 📝 Exercises (fill-in, MCQ, transformation, translation, error-correction,
  speaking, writing, mixed) — **answers after the exercises**

### ➕ NEW mandatory sections (every chapter from Ch.12 onward)
Per the Advanced-Grammar-Extensions spec, every new chapter **also** includes, as standard sections:
```
 1. LEVEL DIFFERENCES (A1→C2)   what to know · acceptable mistakes · new grammar · vocabulary ·
                                 speaking/writing ability · native expectations · IELTS equivalent
 2. NATIVE SPEAKER INSIGHT       how natives really use it · how textbooks simplify it ·
                                 spoken shortcuts · contractions · informal/formal/business/academic
 3. ALTERNATIVE EXPRESSIONS      other structures + formal/informal/native/business/academic/
                                 IELTS/American/British alternatives, with when natives pick each
 4. DEEP ANALYSIS TASKS          Grammar Detective · find & fix the mistake · why is it wrong ·
                                 rewrite naturally · compare meanings · choose the most native version
```
