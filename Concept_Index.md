# 🔎 Concept Index — Where Everything Lives

### *The book's master locator: find any grammar concept in seconds*
#### 🇦🇿 *Kitabın əsas naviqatoru: istənilən qrammatik mövzunu saniyələr içində tap*

> **Azərbaycanlı oxucu üçün:** Bu səhifə bir **indeks**dir (məzmun deyil). Hər mövzunun kitabın
> **harasında** izah olunduğunu, **harada** müqayisə edildiyini, **hansı** fəsildə təkrar işləndiyini,
> **səhvlərinin** və **məşqlərinin** harada olduğunu göstərir. Bir mövzunu axtarırsansa — **əvvəlcə bura bax.**

This is an **index, not a lesson.** For every concept it tells you *where* it is:
**introduced**, **compared**, **reused**, its **common mistakes**, **extra drills**, its **master summary**,
and where it is **finally tested**. Start here whenever you want to jump straight to a topic.

## 🧭 How to read the columns

| Column | Meaning |
|---|---|
| 📖 **Introduced** | The dedicated chapter that teaches the concept from scratch. |
| ⚖️ **Compared** | Where it is set *side-by-side* against a look-alike (duels / charts / mixed review). |
| 🔁 **Reused in** | Later chapters that *apply* the concept inside a bigger structure. |
| ⚠️ **Mistakes** | The common-error family for it in [Ch.39][39]. |
| 🎯 **Drills** | Cross-chapter practice beyond the concept's own quiz. |
| 📊 **Summary** | The one-page master reference. |
| 🏁 **Final** | Where it is assessed at the end. |

> 💡 Every chapter *also* contains its **own** quiz, mistakes list and mini-stories — the columns
> below point to the **cross-chapter** places where the concept reappears.
