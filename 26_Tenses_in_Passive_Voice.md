# 🟪 PART 5 · Chapter 26
# Tenses in the Passive Voice — *"The letter was written."* 🔄
### Active → Passive · be + V3, in every tense

> 🇦🇿 **Bu fəsil haqqında:** Bəzən **kimin** etdiyi yox, **nəyin baş verdiyi** vacibdir.
> *"Oğru velosipedimi oğurladı"* yerinə *"Velosipedim oğurlandı"* deyirik. Bu, **passive voice**
> (məchul növ) adlanır. Yaxşı xəbər: Azərbaycan dilində də passive var — **-ıl/-il** şəkilçisi
> (*yazmaq → yazılmaq, tikmək → tikilmək*). İngiliscə bunu **be + V3** ilə düzəldirik. Bu fəsil
> yeni zaman öyrətmir — zamanları **passive formada** qurmağı öyrədir. 🔁

## 🧭 Legend / İşarələrin mənası
```
 ✅ correct   ❌ wrong   ⚠️ careful   💡 tip   🧠 memory   🇦🇿 Azerbaijani note
 V3 = past participle (written, made, taken, done, eaten)
 "agent" = the doer of the action (the "by..." part)
 ACTIVE = subject DOES it   ·   PASSIVE = subject RECEIVES it
```

---

# 1. 📌 Introduction / Giriş

Every action sentence can be told **two ways**:

```
   ACTIVE:   The chef    cooked   the meal.
             (doer)      (action) (receiver)
             → focus on WHO did it

   PASSIVE:  The meal    was cooked   (by the chef).
             (receiver)  (action)     (doer — optional)
             → focus on WHAT happened
```

To make a passive, you **flip** the sentence:
```
   1. the OBJECT of the active  →  becomes the SUBJECT
   2. add the verb "BE" (in the right tense) + V3 (past participle)
   3. the old subject becomes "by + agent"  (optional!)
```

```
   The chef cooked the meal.
        └──────┐    └──┐
   The meal  was cooked  (by the chef).
```

That's the whole machine: **be + V3.** The only thing that changes between tenses is the **form of
"be."** 🔁

---

# 2. 🤔 Why do we use it? / Niyə işlədirik?

We choose the passive when the **doer** is unknown, obvious, unimportant, or when we want to focus
on the **action/result** instead:

```
   1. DOER UNKNOWN:      My bike was stolen.        (I don't know who)
   2. DOER OBVIOUS:      He was arrested.            (...by the police, of course)
   3. DOER UNIMPORTANT:  The road is being repaired. (who cares who repairs it)
   4. GENERAL "people":  English is spoken here.     (people in general)
   5. FOCUS ON RESULT:   The Mona Lisa was painted in 1503.
   6. FORMAL / SCIENCE:  The sample was heated to 100°C.
   7. DIPLOMATIC:        Mistakes were made.          (avoid naming the blamer 😅)
```

> 🇦🇿 İcraçı naməlum, aydın və ya əhəmiyyətsizdirsə — passive işlədirik. Diqqət **hərəkətə**
> yönəlir, icraçıya yox.

# 4. ⏳ The "be + V3" engine / "be + V3" mexanizmi

The passive is **always** `be + V3`. To change tense, you only change **be**:

```
   ┌────────────────────────────────────────────────┐
   │   PASSIVE  =  [ BE in the tense you want ]  +  V3 │
   └────────────────────────────────────────────────┘

   is/are    + made   →  present simple
   was/were  + made   →  past simple
   is being  + made   →  present continuous
   has been  + made   →  present perfect
   will be   + made   →  future simple
   ...                →  V3 NEVER changes; only "be" moves.
```

> 🧠 Learn the **V3** of common verbs (made, written, taken, done, given, seen, built, sold…) and
> the passive becomes easy — you just bolt them onto the right form of *be*.

# 5. 🧪 Formula / Düstur — passive in every tense

```
 ┌──────────────────────────┬─────────────────────┬──────────────────────────────┐
 │ TENSE                    │ ACTIVE              │ PASSIVE (be + V3)            │
 ├──────────────────────────┼─────────────────────┼──────────────────────────────┤
 │ Present Simple           │ makes / make        │ is / are made                │
 │ Present Continuous       │ is/are making       │ is / are being made          │
 │ Present Perfect          │ has/have made       │ has / have been made         │
 │ Past Simple              │ made                │ was / were made              │
 │ Past Continuous          │ was/were making     │ was / were being made        │
 │ Past Perfect             │ had made            │ had been made                │
 │ Future Simple (will)     │ will make           │ will be made                 │
 │ Future (going to)        │ is going to make    │ is going to be made          │
 │ Future Perfect           │ will have made      │ will have been made          │
 │ Modals (can/must/should) │ can make            │ can be made                  │
 └──────────────────────────┴─────────────────────┴──────────────────────────────┘
```

⚠️ **Rarely used (too clumsy):** Future Continuous passive (*will be being made*) and Perfect
Continuous passives (*has been being made*). Native speakers **avoid** these — reword instead.

### 📝 Basic patterns
```
 ✅ POSITIVE:  The house was built in 1990.
 ❌ NEGATIVE:  The house wasn't built in 1990.
 ❓ QUESTION:  Was the house built in 1990?
 👤 + AGENT:   The house was built by my grandfather.   (by + doer, optional)
```

# 6. 🧱 Sentence Structure / Cümlə quruluşu

```
   ACTIVE:
   ┌──────────┬──────────┬────────────┐
   │ Shakespeare │ wrote │ Hamlet.    │
   │ (subject)   │(verb) │ (object)   │
   └──────────┴──────────┴────────────┘
            │ flip │
            ▼
   PASSIVE:
   ┌────────┬───────────┬──────────────────────┐
   │ Hamlet │ was written │ by Shakespeare.     │
   │(subject)│ (be + V3)  │ (by + agent, opt.)  │
   └────────┴───────────┴──────────────────────┘
```

🔑 Only verbs with an **object** (transitive) can be passive. The object becomes the subject.
> 🇦🇿 Yalnız **obyekti olan** (keçidli) fellər passive ola bilər. Obyekt → mübtəda olur.

---

# 7. 📐 Grammar Rules / Qrammatika qaydaları

1. **Passive = `be` (correct tense) + `V3`.** Only *be* changes for tense. 👉 §5.
2. **The object becomes the subject;** the subject becomes **`by + agent`** (optional).
3. **Use `by` only when the doer matters** (surprising/important). Otherwise drop it.
4. **Intransitive verbs have NO passive** — no object to promote:
   ❌ *was happened, was arrived, was died, was slept.* (happen, arrive, die, come, go, sleep…)
5. **Subject–verb agreement** on *be*: *The book **is** sold* / *The books **are** sold.*
6. **Two-object verbs** (give, send, offer, tell, show) → usually make the **person** the subject:
   *They gave me a prize* → *I **was given** a prize* (or *A prize **was given** to me*).
7. **`get`-passive** (informal): *My phone **got** stolen* = *was stolen.*
8. **Modals:** modal + **be** + V3: *It **can be** done. It **must be** finished.*

```
   QUICK CHECK before making a passive:
   ❓ Does the verb have an OBJECT?  No → can't be passive (it's intransitive).
   ❓ What tense is the ACTIVE verb? → put BE in that tense, add V3.
   ❓ Does the doer matter?          → keep "by..."; if not, drop it.
```

# 8. ✍️ Form Rules — V3 & agreement / Forma qaydaları

| Verb | V3 (past participle) | Passive example |
|---|---|---|
| make | made | It is made of wood. |
| write | written | It was written by hand. |
| take | taken | The photo was taken in 2020. |
| build | built | It was built last year. |
| do | done | It will be done soon. |
| give | given | She was given a medal. |
| break | broken | The window has been broken. |
| sell | sold | The car was sold yesterday. |

> ⚠️ **#1 passive error:** using **V2** instead of **V3**. ❌ *was wrote* → ✅ *was **written***;
> ❌ *was took* → ✅ *was **taken***. The full irregular list is in the Appendix. 🇦🇿 Passive-də
> **V3** (yazılmış forma) işlədilir, V2 yox.

# 9. 🔔 Signal Words / Siqnal sözləri

The passive often appears with:
```
   by + (agent)         ·  was/were + V3        ·  is/are + V3
   "It is said that..."  ·  "It is believed that..."  ·  "It was reported that..."
   process words: first, then, next, after that, finally
   contexts: notices, recipes, instructions, science, news, history
```
> 💡 In recipes/instructions and science, the passive is the **default** voice:
> *"The onions **are chopped**, then **fried**…"* 🇦🇿 Reseptlərdə və elmi mətnlərdə passive əsasdır.

# 10. 🚫 Common Mistakes / Ən çox edilən səhvlər

```
 ❌ The house built in 1990.
 ✅ The house was built in 1990.            (passive needs BE)

 ❌ The letter was wrote by Tom.
 ✅ The letter was written by Tom.           (V3, not V2)

 ❌ The work is done yesterday.
 ✅ The work was done yesterday.             (right tense of "be")

 ❌ An accident was happened.
 ✅ An accident happened.                     (happen is intransitive — no passive)

 ❌ This song is loved by everyone (overuse?).  → often fine, but:
 ❌ The ball was kicked by me.   ✅ I kicked the ball.   (don't passivize when active is clearer)

 ❌ The room cleans every day.
 ✅ The room is cleaned every day.            (it doesn't clean itself!)
```

> 🇦🇿 **Ən böyük səhvlər:** (1) "be" unutmaq; (2) V2 işlətmək (V3 lazımdır); (3) keçidsiz felləri
> passive etmək (*happen, arrive, die* → passive YOX).

# 11. ⚖️ Comparison / Müqayisə

### Active vs Passive — when to use which
```
 ┌──────────┬────────────────────────────────────┬──────────────────────────────┐
 │ VOICE    │ FOCUS                              │ USE WHEN                     │
 ├──────────┼────────────────────────────────────┼──────────────────────────────┤
 │ ACTIVE   │ the DOER ("who did it")            │ the doer is known & important │
 │ PASSIVE  │ the ACTION / RECEIVER             │ doer unknown/obvious/general  │
 │          │                                    │ or formal/scientific/news     │
 └──────────┴────────────────────────────────────┴──────────────────────────────┘
```

### Same fact, two voices
```
   Active:  Picasso painted this.        → focus on Picasso
   Passive: This was painted by Picasso. → focus on the painting
   Passive: My wallet was stolen.        → I don't know who (no "by")
```

### `be`-passive vs `get`-passive
```
   be-passive (neutral/formal):  He was injured in the match.
   get-passive (informal/sudden): He got injured in the match.
   ⚠️ "get" is common in speech but avoided in formal writing.
```

# 12. 🇦🇿 Azerbaijani Translation Tips / Tərcümə məsləhətləri

Good news — Azerbaijani **has** a passive, built with a **suffix**, not a helper verb. 🎉

```
   ENGLISH (be + V3)            AZERBAIJANI (verb + -ıl/-il/-ın/-in)
   It is written.           →   Yazılır.
   It was written.          →   Yazıldı.
   The house was built.     →   Ev tikildi.
   The door was opened.     →   Qapı açıldı.
   English is spoken here.   →   Burada İngiliscə danışılır.
   by Shakespeare           →   Şekspir tərəfindən
```

⚠️ **Three traps for Azerbaijani speakers:**
1. **English needs a separate "be" word; Azerbaijani uses a suffix.** Don't forget the **be**:
   ❌ *The house built in 1990* → ✅ *The house **was** built…*
2. **Use V3, not V2.** AZ has one passive stem; English distinguishes *wrote* (V2) vs *written*
   (V3). Passive = **V3**.
3. **"by" = "tərəfindən"** — but only add it when the doer matters (English drops it more often
   than you'd expect).

> 🇦🇿 Qızıl qayda: **İngiliscə passive = be + V3. Azərbaycanca = fel + -ıl/-il şəkilçisi.
> "tərəfindən" = by.**

# 13. 🔊 Pronunciation Notes / Tələffüz qeydləri
```
   was       → /wɒz/ (strong) ;  weak /wəz/ in speech ("it wəz made")
   were      → /wɜː/ (strong) ;  weak /wə/ ("they wə sold")
   been      → /biːn/ or /bɪn/
   -ed endings on regular V3:  /t/ washed, /d/ cleaned, /ɪd/ painted
   "by"      → /baɪ/  ;  often weak/quick before the agent
```
> 💡 In fast speech, *was/were* shrink to **/wəz/, /wə/**: *"The bridge wəz built in 1900."* Don't
> stress them. 🇦🇿 *was/were* danışıqda zəifləyir.

# 23. 🎤 Speaking Tips / Danışıq məsləhətləri
1. Use the passive when you **don't know or don't care** who did it: *"It got cancelled."*
2. In speech, prefer **get**: *"My laptop got stolen."* (sounds natural)
3. Don't over-passivize — if the doer matters, **active is clearer and stronger**.
4. Master the **V3** forms; that's what makes the passive flow.

# 24. 🖊️ Writing Tips / Yazı məsləhətləri
1. **Formal/academic/news → passive** for objectivity (*The data were analysed*).
2. **Process descriptions** (IELTS Task 1, recipes, manuals) → passive default.
3. But don't drown the reader in passives — mix with active for clarity.
4. Drop **by + agent** when it's obvious; keep it when it's the key information.

# 25. 🎧 Listening Tips / Dinləmə məsləhətləri
1. Listen for **was/were + V3** and **is/are being + V3** — the passive signature.
2. *was/were* shrink to **/wəz, wə/** in fast speech — don't miss them.
3. **get + V3** (*got fired, got stolen*) is a passive in disguise.

# 26. 📖 Reading Tips / Oxu məsləhətləri
1. News & science lean heavily on the passive — the **doer is often hidden** on purpose.
2. *"It is said/believed/reported that…"* signals an **unnamed** source.
3. Spot **by + agent** to find the doer when it *is* given.

# 27. 🗽 Native Speaker Usage / Doğma danışanlar necə işlədir
- Natives use the **passive constantly** in news, science, business, and signs.
- In casual speech, the **get-passive** dominates (*got dumped, got promoted*).
- *"Mistakes were made"* is a famous **blame-dodging** passive — natives use it (often ironically).
- Writers are taught to *prefer active* for punch, but the passive is the right tool when the doer
  is unknown, obvious, or beside the point.

# 28. ❓ FAQ / Tez-tez verilən suallar

**Q1. How do I form the passive?**
**be** (in the tense you want) **+ V3**. Only *be* changes for tense; V3 stays fixed. 👉 §5.

**Q2. When should I use it?**
When the **doer is unknown, obvious, unimportant, or general**, or for **formal/scientific/news**
writing. 👉 §2.

**Q3. Do I always need "by..."?**
No — only when the doer is **important or surprising** (*by Shakespeare*). Usually drop it. 👉 §7.

**Q4. Can every verb be passive?**
No — only **transitive** verbs (with an object). *happen, arrive, die, come, go, sleep* have **no
passive.** 👉 §7.

**Q5. What's the get-passive?**
An informal passive: *"It got broken" = "It was broken."* Common in speech, avoided in formal
writing. 👉 §11.

**Q6. Two objects — which becomes the subject?**
Usually the **person**: *They gave me a prize* → *I was given a prize.* 👉 §7.

**Q7. How does it map to Azerbaijani?**
AZ uses a **suffix** (*-ıl/-il*: yazıl-, tikil-), English uses **be + V3**. *by = tərəfindən.* Don't
forget the *be* and use *V3* (not V2). 👉 §12.

# 📚 MINI STORY — *"The Old Factory"*
### (passive voice in action) 🏭

> 🎯 Read 🇬🇧 first. **Bold** = passive verbs (be + V3). Notice how the passive lets the story focus
> on **what happened to the place**, not on individual doers. Then read 🇦🇿 and do the questions.

## 🇬🇧 The Old Factory

At the edge of town, an old factory **had been abandoned** for thirty years.

It **was built** in 1950, back when the whole town worked there. Shoes **were made** there — good,
strong shoes that **were sold** all over the country. Then, one cold winter, the factory **was
closed**, and hundreds of workers **were let go**.

For years, nothing happened. The windows **were broken** by children. The walls **were covered**
in graffiti. People said the place **was haunted**.

Then, last spring, everything changed. The building **was bought** by a young woman named Nigar.
"It can **be saved**," she told the local paper. "It shouldn't **be torn down**."

The roof **was repaired**. The old machines **were removed**. The brick walls **were cleaned** and
the broken windows **were replaced**. Slowly, the factory **was transformed** into something new.

Today, the old factory **is filled** with light. Coffee **is served** where shoes were once
packed. Art **is displayed** on the cleaned walls. Concerts **are held** in the main hall.

On opening night, an old man stood quietly in the corner. He **had been employed** there fifty
years ago, when he was just a boy.

"This place **was given** a second life," he whispered. "Just like me."

Nigar smiled. Some things, she thought, are not destroyed by time. They are only **waiting to be
remembered.**

| 🇬🇧 From the story | Tense of the passive | be + V3 |

# 🧩 EXPANSION PACK — *Passive Voice* (E1–E12)
### The advanced passives: reporting, causative, infinitives. ☕

> 🎯 §1–§29 gave you "be + V3" in every tense. This pack adds the **high-value advanced passives**
> (the reporting passive, the causative *have something done*, passive infinitives) and the AZ
> traps. These are the structures that separate B1 from C1. Read slowly.

---

## E1 · The REPORTING passive — *"It is said that…"* 📰
When we report what **people in general** say/think/believe, we use two passive structures:

```
 STRUCTURE A:  It + be + V3 + that-clause
    It is said that he is very rich.
    It was believed that the earth was flat.
    It has been reported that prices will rise.

 STRUCTURE B:  Subject + be + V3 + to-infinitive
    He is said to be very rich.
    The earth was believed to be flat.
    Prices are reported to be rising.
```
🔑 Verbs that work this way: *say, believe, think, know, report, expect, consider, claim, allege,
understand.*
```
   "to be" (same time):     He is thought to be rich.       (= that he IS rich)
   "to have been" (earlier): He is thought to have been rich. (= that he WAS rich)
```
> 💡 This is **gold for IELTS/academic writing** — it lets you state an opinion **impersonally**.
> 🇦🇿 *It is said that…* = "Deyilir ki…"; *He is said to be…* = "Onun … olduğu deyilir."

---

## E2 · Two objects → two possible passives 🎁
Verbs like *give, send, offer, show, tell, pay, lend, award, teach* have **two objects** (a person
+ a thing). Either can become the subject — but the **person** version is far more common.

```
   Active:   They gave  [me]  [a prize].
                       person  thing
   Passive 1 (person): I was given a prize.            ✅ most natural
   Passive 2 (thing):  A prize was given to me.        ✅ note "to"
```
More: *"She was offered a job." · "We were told a secret." · "He was sent an email." · "They were
paid a bonus."*
> 🧠 Default to making the **person** the subject — it sounds the most English. 🇦🇿 İki obyektli
> fellərdə adətən **şəxsi** mübtəda et: *I was given…*

---

## E3 · Passive infinitives & -ing forms 🔧
The passive isn't only a full verb — it appears as **to be + V3** and **being + V3**:

```
   to be + V3 (passive infinitive):
      The form needs to be signed.
      I want to be left alone.
      This has to be done today.

   being + V3 (passive -ing):
      I hate being told what to do.
      She remembers being chosen for the team.
      He avoided being seen.
```
> 💡 After *need, want, have, has, would like* → **to be + V3**. After *hate, love, avoid, remember,
> mind, enjoy* → **being + V3**. 🇦🇿 *to be done* (ediləsi) / *being told* (deyilmək).

---

## E4 · CAUSATIVE — *have / get something done* 🛠️
This is **not** the ordinary passive — it means **you arrange for someone else to do it for you.**

> 📚 **Related —** this is a preview; the causative has its own full chapter with *make / let / help* and all its quirks: **[Chapter 30 — The Causative](30_Causative.md)**. 🇦🇿 *Bu qısa baxışdır; kauzativin tam fəsli (make/let/help daxil) Ch.30-dadır.*

```
   STRUCTURE:  have / get  +  [thing]  +  V3
   I had my hair cut.            (a hairdresser cut it — I arranged it)
   We're having the house painted.
   She got her car repaired.
   I need to get my passport renewed.
```
Compare the meanings:
```
   I cut my hair.        → I did it myself.
   I had my hair cut.    → someone did it for me (I arranged/paid).  ← causative
   My hair was cut.      → plain passive (no "I arranged it" meaning)
```
⚠️ Also for **bad things that happen to you**: *"He had his wallet stolen." / "She got her phone
hacked."*
> 🧠 **have/get + thing + V3 = "I arranged it" OR "it happened to me."** A super-common everyday
> structure. 🇦🇿 *saçımı kəsdirdim* = I had my hair cut (başqası kəsdi, mən təşkil etdim).

---

## E5 · The `get`-passive — when it works 🔀
*get*-passive is informal and mostly for **sudden changes or events** (not states):

```
 ✅ got married, got divorced, got dressed, got lost, got stuck, got hurt, got fired,
    got promoted, got caught, got broken, got stolen
 ⚠️ avoid in formal writing → use "be"
 ❌ "The house gets cleaned by a robot" sounds odd for a calm fact → "is cleaned" is better
```
> 💡 *get* adds a feeling of **something happening TO you**, often unexpectedly/quickly. 🇦🇿 *get* +
> V3 → qəfil/dəyişiklik bildirən hadisələr: *got married (evləndi), got lost (azdı).*

## E6 · `by` vs `with` — doer vs tool 🔪
```
   by  + the DOER (agent):      He was bitten BY a dog.
   with + the TOOL (instrument): The bread was cut WITH a knife.
   covered with / in:            The floor was covered with/in dust.
   filled with:                  The jar was filled WITH water.
```
🔑 **by** = who/what *did* it; **with** = what it was *done using* (the instrument/material).
> 🇦🇿 *by* = icraçı (tərəfindən); *with* = alət (ilə). *bıçaqla kəsildi = cut with a knife.*

---

## E7 · Passive of PHRASAL VERBS — keep the particle! 🧲
The particle (off/up/after/into…) **stays** attached in the passive:

```
   They called off the match.   → The match was called off.
   Someone broke into the house. → The house was broken into.
   A nurse looked after him.      → He was looked after by a nurse.
   They will deal with it.        → It will be dealt with.
```
⚠️ Don't drop the particle: ❌ *The match was called.* → ✅ *was called **off**.*
> 🇦🇿 Frazal fellərdə hissəcik (off/up/into) passive-də də **qalır**.

---

## E8 · Verbs that RESIST the passive 🚫
Beyond intransitives (§7), some **transitive** verbs still avoid the passive — mostly **state
verbs**:

```
   have (= own):   ❌ A car is had by me.   ✅ I have a car.
   fit / suit:     ❌ I am suited by this.   ✅ This suits me.
   resemble:       ❌ She is resembled by me. ✅ I resemble her.
   lack, cost, weigh, equal: ❌ usually no passive.
```
> 🧠 If the verb describes a **state/relationship** (not a real action done TO something), it
> usually can't be passive. 🇦🇿 Hal/münasibət bildirən fellər (*have, fit, suit, resemble*) passive
> olmur.

## E9 · Lexicalized passives — *born*, and friends 👶
Some passives are **fixed expressions** you just memorize:

```
   I was born in 2001.            (never "I bore" / "I am born" for date of birth)
   She is based in London.
   He is said to be... (E1)
   I'm done. / It's finished.     (passive-feeling states)
   be stuck / be lost / be gone   (states from past events)
```
> 💡 *"I was born"* is the most common passive most learners get wrong — it's **always passive**.
> ❌ *I born in… / I am born in…* → ✅ *I **was born** in…* 🇦🇿 *anadan olmuşam* → *I was born.*

## E10 · 🗺️ THE MASTER DECISION MAP — active or passive?
```
 ┌─────────────────────────────────────────────────────────────────────────────┐
 │  START: I have an action sentence. Active or passive?                        │
 └─────────────────────────────────────────────────────────────────────────────┘
        │
        ▼
   Is the DOER important / known / interesting?
        ├── YES → use ACTIVE        "Picasso painted it."
        └── NO ↓  (unknown / obvious / general / I want focus on the action)
                 │
                 ▼
            Does the verb have an OBJECT? (transitive?)
                 ├── NO → can't be passive (happen, arrive, die...) → keep active
                 └── YES ↓
                          │
                          ▼
                   BUILD THE PASSIVE:
                   1. object → subject
                   2. BE in the active's tense + V3
                   3. add "by + doer" ONLY if it matters
                   ─────────────────────────────────────
                   Special cases:
                   • "people say/think" → It is said that… / S is said to… (E1)
                   • "I arranged it"     → have/get + thing + V3 (causative, E4)
                   • informal/sudden     → get + V3 (E5)
```
🇦🇿 **Açar sual:** *İcraçı vacibdirmi?* Yox → passive. *Obyekt varmı?* Yox → passive olmaz.


## E12 · 🇦🇿 Azerbaijani-interference mistakes (deep cut)
```
 ❌ I born in Baku in 2000.
 ✅ I was born in Baku in 2000.        → "anadan olmuşam" həmişə passive: was born.

 ❌ The house built in 1990.
 ✅ The house was built in 1990.        → "be" unutma.

 ❌ The letter was wrote by him.
 ✅ The letter was written by him.       → V3 (written), V2 (wrote) yox.

```

# 📖 MINI STORY #2 — *"The Bottle on the Shore"*
### (passive voice in action) 🌊

> 🎯 Read 🇬🇧 first. **Bold** = passive verbs (be + V3). Notice how the passive keeps the mystery —
> we often **don't know who** did each thing. Then read 🇦🇿 and do the questions.

## 🇬🇧 The Bottle on the Shore

The bottle **was found** by a girl named Aysu, half-buried in the wet sand.

Inside, a folded paper **had been sealed** with wax. When it **was opened**, a message **was
revealed**, written in faded blue ink.

*"If this is ever read,"* it began, *"please know that I was not afraid."*

The letter **had been written** by a sailor. His ship, the *Umid*, **had been caught** in a storm
forty years earlier. The bottle **had been thrown** into the sea on the last night, when all hope
**was lost.**

Aysu took it to the old harbour museum. There, the message **was examined** by an expert.

"This is remarkable," he said. "The *Umid* **is still talked about** here. It **was believed to
have been destroyed** with no survivors. But one man **was rescued** — he **was pulled** from the
water by a fishing boat. His name **is engraved** on a plaque by the door."

Aysu looked. The name **was carved** in stone: *Captain Rashid Mehdiyev.*

"He's still alive," the expert added quietly. "He's ninety-one. He **is cared for** at a home up
the coast."

A week later, the bottle **was returned** to its writer.

The old captain held it in shaking hands. He had thrown it into the dark, certain that it **would
never be found.**

"I was told the sea kept nothing," he whispered. "But it seems some things are only **kept** — to
be **given back** when they are needed most."

The bottle **is now displayed** in the museum. Beside it, a new line **has been added** to the
plaque: *The message was received.*

```
 │ modal: can/must be made  ·  get-passive: got made (informal)        │
 │ 🇦🇿 fel + -ıl/-il   ·   by = tərəfindən                            │
 └──────────────────────────────────────────────────────────────────┘
```
