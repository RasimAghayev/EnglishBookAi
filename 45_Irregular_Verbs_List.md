## 🧭 Legend / İşarələrin mənası
```
 V1 = base form (go)         used in: present, future, infinitive, after modals
 V2 = past simple (went)     used in: past simple
 V3 = past participle (gone) used in: perfect tenses (have/had + V3) & passive (be + V3)
 IPA = pronunciation guide   ·  🇦🇿 = Azerbaijani meaning
```

# 1. 📌 Why irregular verbs matter / Niyə vacibdir?

Most English verbs are **regular** — add **-ed** for V2 and V3:
```
   work → worked → worked      play → played → played      clean → cleaned → cleaned
```

But about **180 common verbs are irregular** — their V2 and V3 don't follow the -ed rule:
```
   go → went → gone      see → saw → seen      buy → bought → bought
```

⚠️ The cruel twist: the **most frequent verbs in English are irregular** (be, have, do, go, say,
make, take, come, see, get…). You **cannot avoid** them. The good news — you only need the **two
extra forms (V2 and V3)** for each, and they fall into **predictable patterns.**

> 🇦🇿 İngiliscənin ən çox işlənən felləri məhz qaydasızdır — onları öyrənmək məcburidir, amma
> qruplarla asanlaşır.

# 2. 📖 How to read the table / Cədvəli necə oxumaq

```
   V1 (base)   V2 (past)   V3 (participle)   Example
   ─────────   ─────────   ───────────────   ──────────────────────────────
   go          went        gone              I go · I went · I have gone
   write       wrote       written           I write · I wrote · I have written
```

**Where each form is used:**
```
   V1 → Present Simple, Future (will + V1), after modals (can go), infinitive (to go)
   V2 → Past Simple ONLY:  "I went home."
   V3 → Perfect tenses:    "I have/had/will have gone."
        Passive voice:     "It was written / is done / has been made."
```

> ⚠️ **The #1 irregular-verb error:** mixing up V2 and V3.
> ❌ *I have went* / *I had wrote* → ✅ *I have **gone*** / *I had **written***.
> 🧠 **After have/had/be → ALWAYS V3.** After nothing (plain past) → V2.

# 3. 🗂️ THE 4 SHAPE-GROUPS — learn by pattern, not one-by-one

Irregular verbs sort into **four shapes**. Grouping them makes memorising far easier. 🔑

### 🟦 GROUP 1 — All THREE the same (V1 = V2 = V3)
*The easy ones — nothing changes!*
```
   cut   → cut   → cut          put   → put   → put
   hit   → hit   → hit          let   → let   → let
   set   → set   → set          shut  → shut  → shut
   cost  → cost  → cost         hurt  → hurt  → hurt
   read  → read  → read         (⚠️ spelling same, but sound changes: /riːd/ → /red/ → /red/)
```
> 🧠 "These never change — what you see is what you get." 🇦🇿 Heç dəyişmir.

### 🟩 GROUP 2 — V2 = V3 (last two the same)
*The biggest group — learn ONE extra form and it does double duty.*
```
   buy    → bought  → bought         bring  → brought → brought
   think  → thought → thought        catch  → caught  → caught
   teach  → taught  → taught         find   → found   → found
   make   → made    → made           have   → had     → had
   say    → said    → said           pay    → paid    → paid
   tell   → told    → told           sell   → sold    → sold
   sit    → sat     → sat            stand  → stood   → stood
   win    → won     → won            send   → sent    → sent
   build  → built   → built          feel   → felt    → felt
   keep   → kept    → kept           sleep  → slept   → slept
   leave  → left    → left           lose   → lost    → lost
   meet   → met     → met            hear   → heard   → heard
```
> 🧠 "Past and participle are twins." 🇦🇿 V2 və V3 eynidir → bir əlavə forma öyrən.

### 🟨 GROUP 3 — ALL THREE different (V1 ≠ V2 ≠ V3)
*The "real" irregulars — the famous ones. Worth memorising as a chant.*
```
   go     → went    → gone           do     → did     → done
   see    → saw     → seen           eat    → ate     → eaten
   give   → gave    → given          take   → took    → taken
   write  → wrote   → written        drive  → drove   → driven
   speak  → spoke   → spoken         break  → broke   → broken
   choose → chose   → chosen         freeze → froze   → frozen
   drink  → drank   → drunk          sing   → sang    → sung
   swim   → swam    → swum           begin  → began   → begun
   ring   → rang    → rung           fly    → flew    → flown
   grow   → grew    → grown          know   → knew    → known
   throw  → threw   → thrown         draw   → drew    → drawn
   wear   → wore    → worn           tear   → tore    → torn
   fall   → fell    → fallen         shake  → shook   → shaken
```
> 🧠 "Three faces — say all three aloud, again and again." 🇦🇿 Üçü də fərqli — zəncir kimi əzbərlə.

### 🟧 GROUP 4 — V1 = V3 (first and last the same)
*A small, special group.*
```
   come   → came   → come           become → became  → become
   run    → ran    → run
```
> 🧠 "It comes back to where it started." 🇦🇿 V1 və V3 eynidir.

# 4. 🎵 SOUND-PATTERN FAMILIES — rhyming groups

Many irregulars **rhyme** and change the same way. Learn one, get several free:

```
 🎵 i → a → u
   sing → sang → sung · ring → rang → rung · drink → drank → drunk ·
   swim → swam → swum · begin → began → begun · sink → sank → sunk

 🎵 -ought / -aught (Group 2)
   buy → bought · think → thought · bring → brought · fight → fought ·
   catch → caught · teach → taught

 🎵 ow → ew → own
   grow → grew → grown · know → knew → known · throw → threw → thrown ·
   blow → blew → blown · fly → flew → flown (slight variation)

 🎵 -ear → -ore → -orn
   wear → wore → worn · tear → tore → torn · (bear → bore → borne)

 🎵 -ake → -ook → -aken
   take → took → taken · shake → shook → shaken · (make → made → made — exception!)

 🎵 -ide → -ode → -idden / -ive → -ove → -iven
   ride → rode → ridden · drive → drove → driven · write → wrote → written
```

> 💡 Learning by **rhyme family** is far faster than one-by-one. Chant them: *"sing-sang-sung,
> ring-rang-rung, drink-drank-drunk!"* 🇦🇿 Qafiyə qrupları ilə öyrən — daha sürətli.

# 7. ⚠️ TRICKY PAIRS — the verbs that trip everyone up

### `lie` vs `lay` (the famous one)
```
   lie (recline, no object):  lie → lay → lain     "I lie down. Yesterday I lay down."
   lay (put sth, +object):    lay → laid → laid    "I lay the book on the table. I laid it down."
   lie (tell untruth):        lie → lied → lied    (REGULAR!) "He lied to me."
```
⚠️ Confusing because *lay* is **both** a verb **and** the past of *lie*. 🇦🇿 *lie* (uzanmaq) ≠
*lay* (qoymaq) ≠ *lie* (yalan demək).

### `rise` vs `raise`
```
   rise (go up, no object):   rise → rose → risen   "The sun rises."
   raise (lift sth, +object): raise → raised → raised (REGULAR) "Raise your hand."
```

### `find` vs `found`
```
   find (discover):  find → found → found
   found (establish): found → founded → founded (REGULAR!) "The city was founded in 1900."
```

### Same spelling, different sound
```
   read  → read   → read    /riːd/ → /red/ → /red/   (spelling identical, sound changes!)
   lead  → led    → led      /liːd/ → /led/ → /led/   (spelling changes here)
```

### `hang`
```
   hang (objects): hang → hung → hung      "I hung the picture."
   hang (execute): hang → hanged → hanged  (REGULAR, rare/grim)
```

> 🧠 These six pairs cause most "advanced" irregular-verb errors. Mark this page. 🇦🇿 Bu cütlüklər
> ən çox səhvə səbəb olur.

# 8. 🧠 MEMORY TRICKS / Yadda saxlama üsulları
```
 🧠 1. After HAVE / HAS / HAD / BE → always V3.
        ✅ have GONE · had WRITTEN · was TAKEN.   (never V2 here)

 🧠 2. Chant the rhyme families:
        "sing-sang-sung, ring-rang-rung, drink-drank-drunk!"
        "buy-bought, think-thought, bring-brought, catch-caught!"

 🧠 3. Group 1 = no change (cut-cut-cut). Group 2 = twins (V2=V3).
       Group 3 = three faces. Group 4 = comes back (come-came-come).

 🧠 4. Learn the verb WITH all three forms, never just V1.
        Don't learn "go" — learn "go-went-gone" as ONE unit.

 🧠 5. Make flashcards: front = V1, back = V2 + V3 + 🇦🇿. Test both directions.

 🧠 6. The deadly pair: V2 (plain past) vs V3 (after have/be). Drill it daily.
```

## E2 · DUAL-FORM verbs — two correct answers ✌️
Some verbs have **both** an irregular **-t** form and a regular **-ed** form. Both are accepted
(British often prefers **-t**, American **-ed**):

```
   burn   → burnt / burned        learn  → learnt / learned
   dream  → dreamt / dreamed      spell  → spelt / spelled
   smell  → smelt / smelled       spill  → spilt / spilled
   spoil  → spoilt / spoiled      leap   → leapt / leaped
   kneel  → knelt / kneeled       lean   → leant / leaned
   light  → lit / lighted         speed  → sped / speeded
   prove  → proved → proven/proved  show → showed → shown/showed*
```
> 💡 Don't worry which to pick — **both are right.** Just be **consistent** within one piece of
> writing. (*show* and *prove* are regular in V2 but keep an irregular V3 option: *shown,
> proven*.) 🇦🇿 İki forma da düzgündür — bir mətndə **eyni** formanı işlət.

## E3 · PREFIXED verbs follow their base — one rule, dozens of verbs 🔑
If you know the base verb, you know **every** prefixed version. They change **identically**:

```
   come → came → come     ⇒  become → became → become
   stand → stood → stood   ⇒  understand → understood → understood ·
                              withstand → withstood → withstood
   take → took → taken     ⇒  mistake → mistook → mistaken ·
                              overtake → overtook → overtaken · undertake → undertook → undertaken
   get → got → got         ⇒  forget → forgot → forgotten · beget → begot → begotten
   see → saw → seen        ⇒  foresee → foresaw → foreseen · oversee → oversaw → overseen
   write → wrote → written  ⇒  rewrite → rewrote → rewritten · underwrite → underwrote → underwritten
   go → went → gone         ⇒  undergo → underwent → undergone · forgo → forwent → forgone
   draw → drew → drawn      ⇒  withdraw → withdrew → withdrawn · overdraw → overdrew → overdrawn
   hold → held → held       ⇒  uphold → upheld → upheld · withhold → withheld → withheld
```
> 🧠 Learn the **base**, and all its prefixed cousins come free. 🇦🇿 Əsas feli bil → bütün
> önşəkilçili formaları eyni dəyişir.

## E4 · "False friends" — verbs that LOOK irregular but are REGULAR ⚠️
Don't over-apply patterns! These are **regular** (just add -ed), even though learners often
"irregularize" them:

```
 ✅ want → wanted          ❌ wanted as "went"
 ✅ work → worked
 ✅ walk → walked
 ✅ help → helped
 ✅ play → played
 ✅ open → opened
 ✅ rain → rained
 ✅ start → started

 ⚠️ And the OPPOSITE trap — these ARE irregular, don't add -ed:
 ❌ teached → ✅ taught     ❌ buyed → ✅ bought     ❌ catched → ✅ caught
 ❌ knowed → ✅ knew        ❌ thinked → ✅ thought   ❌ bringed → ✅ brought
 ❌ goed → ✅ went          ❌ comed → ✅ came        ❌ eated → ✅ ate
```
> 🧠 When in doubt, **check the list** — don't guess by sound. 🇦🇿 Səslə təxmin etmə — siyahıya bax.

## E5 · The "-ed" pronunciation rule (for REGULAR verbs) 🔊
Since you're working on past forms, master how regular **-ed** sounds. It has **three**
pronunciations, decided by the **last sound** of the base verb:

```
 ┌───────────┬─────────────────────────────────┬──────────────────────────────┐
 │ -ed sounds│ when base ends in...            │ examples                     │
 ├───────────┼─────────────────────────────────┼──────────────────────────────┤
 │  /ɪd/     │ a /t/ or /d/ sound              │ wanted, needed, started, ended│
 │  /t/      │ a voiceless sound (p,k,f,s,sh,ch)│ stopped, worked, washed, kissed│
 │  /d/      │ a voiced sound (everything else) │ played, cleaned, loved, rained│
 └───────────┴─────────────────────────────────┴──────────────────────────────┘
```
> 💡 Only **/t/** and **/d/** endings add a whole extra syllable (/ɪd/): *want-ED, need-ED.* The
> others **don't** — *worked* = one syllable (/wɜːkt/), not "work-ed." 🇦🇿 *-ed* üç cür səslənir;
> yalnız *t/d*-dən sonra ayrı heca olur.

## E6 · Past participles (V3) as ADJECTIVES — V3's second life 🎨
V3 isn't only for perfect tenses and passives — it works as an **adjective** describing a noun:

```
   a broken window        sınmış pəncərə
   frozen food            dondurulmuş ərzaq
   a written agreement    yazılı razılaşma
   fallen leaves          tökülmüş yarpaqlar
   a stolen car           oğurlanmış maşın
   a hidden treasure      gizli xəzinə
   burnt toast            yanmış tost
   spoken English         danışıq İngiliscəsi
   a well-known author    tanınmış müəllif
   lost property          itirilmiş əşyalar
```
> 🧠 Many adjectives you already use are secretly **V3 forms**: *broken, frozen, hidden, known,
> spoken.* This is why learning V3 pays off everywhere. 🇦🇿 V3 həm də sifət kimi işlənir: *broken,
> frozen, written.*

## E7 · Irregular verbs inside PHRASAL VERBS 🧲
The irregular form **stays irregular** inside a phrasal verb — across all tenses:

```
   break down → broke down → (has) broken down     "The car has broken down."
   give up    → gave up    → (has) given up          "He has given up smoking."
   take off   → took off   → (has) taken off          "The plane had taken off."
   come across→ came across→ (has) come across         "I came across an old photo."
   put up with→ put up with → (has) put up with        "She has put up with a lot."
   make up    → made up    → (has) made up             "They have made up."
```
> 💡 The **particle** (down/up/off) never changes; only the **verb** takes V2/V3. 🇦🇿 Frazal feldə
> yalnız fel hissəsi V2/V3 olur; hissəcik dəyişmir.

## E9 · HOW to memorise efficiently 🧠📚

## E12 · 🇦🇿 Azerbaijani-interference mistakes (deep cut)
```
 ❌ I have went / I have wrote.
 ✅ I have GONE / I have WRITTEN.        → have/has/had-dan sonra V3, V2 yox.

 ❌ He knowed / She teached / They buyed.
 ✅ He knew / She taught / They bought.   → bu fellər qaydasızdır; -ed əlavə etmə.

 ❌ The letter was wrote / The window was broke.
 ✅ The letter was WRITTEN / was BROKEN.  → passive-də V3.

 ❌ I have saw him yesterday.
 ✅ I saw him yesterday.                   → həm V2/V3 qarışıb, həm vaxt: "yesterday" → Past Simple.

 ❌ a broke window / a froze lake.
 ✅ a BROKEN window / a FROZEN lake.        → sifət kimi V3 işlənir.
```
> 🧠 The 4 deep AZ traps: **(1)** V2 instead of V3 after have/be; **(2)** adding -ed to irregulars;
> **(3)** wrong form in passive; **(4)** using V2/V3 as an adjective wrongly (use V3: *broken*).

# 📖 MINI STORY — *"The Night the River Rose"*
### (an irregular-verb showcase) 🌊

> 🎯 This story is **deliberately packed with irregular verbs** — almost every action verb is one.
> **Bold** = an irregular verb (V2 or V3). After reading, use the key to check every form. Read 🇬🇧
> first, then 🇦🇿, then the questions.

## 🇬🇧 The Night the River Rose

It **began** with rain. By midnight, the river **had risen** higher than anyone **had seen** before.

Murad **woke** his little sister. "Get up," he **said**. "We have to **leave**." She **clung** to
her blanket, but he **took** her hand and they **ran**.

The water **had already crept** into the kitchen. Murad **caught** the dog, **threw** open the back
door, and they **fled** into the dark. He **had forgotten** his shoes, but there **was** no time.

They **swam** where the street **had become** a river. Murad **held** his sister tight. Something
**struck** his leg — a branch, swept down by the current. He **felt** the pain but **said**
nothing. He **knew** that if he **let** go, she **would** be lost.

A light **shone** ahead. A boat. A man **threw** them a rope, and they **caught** it. Strong hands
**drew** them up. They **had made** it.

By morning, the rain **had stopped.** The family **stood** on the hill and **looked** down at what
the water **had taken** — the house they **had built**, the garden they **had grown**, the life
they **had known.**

But they **had kept** each other.

"We **lost** everything," his sister **wept.**

"No," Murad **said.** He **held** the dog, who **had shaken** himself dry and **begun** to bark at
the sun. "We **lost** things. Things can be **rebuilt.** We **kept** what **costs** the most."

The river, in time, **fell** back. And the family, in time, **rose** again.

## 🇦🇿 Çayın Daşdığı Gecə *(translation)*

Murad balaca bacısını oyatdı. "Qalx," dedi. "Getməliyik." Bacısı yorğanından yapışdı, amma o, əlindən
tutdu və qaçdılar.

Su artıq mətbəxə girmişdi. Murad iti tutdu, arxa qapını taybatay açdı və qaranlığa qaçdılar. Ayaqqabılarını
unutmuşdu, amma vaxt yox idi.

Küçənin çaya çevrildiyi yerdə üzdülər. Murad bacısını möhkəm tutdu. Nəsə ayağına dəydi — axının
gətirdiyi bir budaq. Ağrını hiss etdi, amma heç nə demədi. Bilirdi ki, əlini buraxsa, bacısı itər.

Qarşıda bir işıq parladı. Bir qayıq. Bir kişi onlara kəndir atdı, onlar da onu tutdular. Güclü əllər
onları yuxarı çəkdi. Nail olmuşdular.

Səhərə yağış kəsmişdi. Ailə təpədə dayanıb suyun apardığına baxdı — tikdikləri ev, becərdikləri bağ,
yaşadıqları həyat.

Amma bir-birlərini saxlamışdılar.

"Hər şeyimizi itirdik," bacısı ağladı.

"Yox," dedi Murad. Silkələnib qurumuş və günəşə hürməyə başlamış iti tutdu. "Əşyaları itirdik. Əşyalar
yenidən qurula bilər. Ən baha olanı saxladıq."

Çay, zaman keçdikcə, geri çəkildi. Ailə də, zaman keçdikcə, yenidən ayağa qalxdı.
