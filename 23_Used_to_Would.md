# 1. 📌 Introduction / Giriş

Some things in our past are **finished**. You don't do them anymore. Maybe you **used to** ride a
bike to school. Maybe there **used to be** a small shop on your street. Maybe, every summer, your
family **would** travel to the village.

This chapter teaches **two tools** for talking about the past that is **gone but remembered**:

```
 ┌─────────────────────────────────────────────────────────────────┐
 │   used to + V1   →  past HABITS and past STATES (no longer true) │
 │   would + V1     →  repeated past ACTIONS only (nostalgic)       │
 └─────────────────────────────────────────────────────────────────┘
```

> 🇦🇿 İkisi də Azərbaycanca çox vaxt **"-ardı / -ərdi"** ilə tərcümə olunur:
> *I used to swim* = *Mən üzərdim.*  ·  *We would go* = *Biz gedərdik.*

These are not "advanced grammar monsters." They are **friendly, everyday English**. Native
speakers use them constantly when they remember childhood, old jobs, old cities, ex-habits.

---

# 2. 🤔 Why do we use them? / Niyə işlədirik?

Past Simple just says **something happened once**:
> *I played football yesterday.* (one time / bir dəfə)

But life has **patterns** that repeated for years and then **stopped**. To show "this was my
normal life **before**, but **not now**," we need a special tool:

> *I **used to** play football every weekend.* (a repeated habit, now finished)
> 🇦🇿 *Mən hər həftəsonu futbol **oynayardım**.* (təkrarlanan, indi bitmiş)

The hidden message of *used to / would* is always:

```
   "This WAS true in the past.   ...but it is NOT true now."
   Bu keçmişdə belə idi.          ...amma indi belə deyil.
```

That little **contrast with now** is the whole reason these forms exist. 💡

# 4. ⏳ Timeline / Zaman xətti

`used to / would` lives in the past and **stops before now**:

```
   PAST  (repeated, normal for a long time)              NOW
   ●──●──●──●──●──●──●──●──●──●──●──● ──────────✕────────▶  time
   |  I used to play / I would play  |          | not anymore |
   └──── a long stretch of habit ────┘          └── it stopped ─┘

   🇦🇿  keçmişdə uzun müddət təkrarlanırdı  →  indi olmur
```

Compare with **Past Simple** (one dot):

```
   Past Simple:   ────────────●──────────────▶   "I played yesterday." (once)
   used to:       ──●─●─●─●─●─────────────────▶   "I used to play." (many times, finished)
```

# 5. 🧪 Formula / Düstur

### 🟢 `used to`

```
 ✅ POSITIVE
    Subject + used to + V1
    I / You / He / She / It / We / They  used to  live  in Baku.
    → Mən/Sən/O ... Bakıda yaşayardım/yaşayardı.

 ❌ NEGATIVE   (⚠️ after "did", it becomes "use to" — NO "d")
    Subject + didn't + use to + V1
    I didn't use to like coffee.        → Mən qəhvəni sevmәzdim.
    (also possible: never used to →  I never used to like coffee.)

 ❓ QUESTION   (⚠️ again "use to", NO "d")
    Did + subject + use to + V1 ?
    Did you use to play the piano?      → Sən pianoda çalardınmı?

 ❓❌ NEGATIVE QUESTION
    Didn't + subject + use to + V1 ?
    Didn't you use to live here?        → Sən burada yaşamırdınmı (əvvəllər)?

 💬 SHORT ANSWERS
    Yes, I did.   /   No, I didn't.
```

> ⚠️ **The #1 spelling trap:** With **did / didn't**, drop the **-d**:
> ✅ *Did you **use to**…?* · ✅ *I didn't **use to**…* — NOT *~~did you used to~~*.
> (In speech "use to" and "used to" sound the same — /juːs tə/ — so this trap is invisible
> until you **write** it.)

### 🔵 `would` (habits only)

```
 ✅ POSITIVE
    Subject + would + V1
    Every winter, we would build a snowman.   → Hər qış adam düzəldərdik.

 ❌ NEGATIVE   (⚠️ "wouldn't" can sound like "refused" — use carefully)
    As a child, I wouldn't eat vegetables.    → Uşaqlıqda tərəvəz yeməzdim.

 ❓ QUESTION   (rare for habit; usually we ask with "used to")
    Did you use to…?  ← preferred for questions
```

> ⚠️ **`would` needs a past-time signal** to be clear (*every summer, back then, when I was
> young, in those days*). Without it, "would" can be misread as a conditional ("I would go" = if…).
> `used to` needs **no** such signal — it stands alone.

# 6. 🧱 Sentence Structure / Cümlə quruluşu

```
 ┌──────────┬──────────┬───────┬──────────────────────────────┐
 │ SUBJECT  │ used to  │  V1   │   rest                       │
 ├──────────┼──────────┼───────┼──────────────────────────────┤
 │   She    │ used to  │ walk  │   to work every day.         │
 │   We     │ used to  │ have  │   a big garden.              │
 ├──────────┼──────────┼───────┼──────────────────────────────┤
 │ SUBJECT  │  would   │  V1   │   rest                       │
 │   He     │  would   │ sit   │   by the window for hours.   │
 └──────────┴──────────┴───────┴──────────────────────────────┘
```

🔑 The verb after **used to** / **would** is **always V1 (base form)** — never V-ing, never V2.
> ✅ used to **go** · ❌ used to ~~going~~ · ❌ used to ~~went~~

---

# 7. 📐 Grammar Rules / Qrammatika qaydaları

1. **`used to` covers BOTH actions and states.**
   - Action: *I used to run.* (habit)
   - State: *I used to have long hair.* / *There used to be a tree here.*
2. **`would` covers ONLY repeated actions — NOT states.** ⚠️
   - ✅ *We would visit Granny.* (action)
   - ❌ *We ~~would have~~ a dog.* → use ✅ *We used to have a dog.* (state = possession)
   - ❌ *I ~~would be~~ shy.* → ✅ *I used to be shy.* (state = be)
3. **State verbs block `would`:** *be, have (own), like, know, believe, want* → use **used to**.
4. **After did/didn't → "use to"** (drop the -d).
5. **`would` prefers a set scene** (a past time frame given first).
6. For something **still true now**, use **Present Simple**, NOT used to:
   - ❌ *I used to live in Baku* (if you STILL live there) → ✅ *I live in Baku.*

```
   QUICK TEST:  Is it a STATE (be / have / like / know)?
        YES → only "used to"   ✅
        NO (it's an action)    → "used to" OR "would"  ✅✅
```

# 8. ✍️ Spelling & Form Rules / Yazılış qaydaları

| Form | ✅ Correct | ❌ Wrong |
|---|---|---|
| Positive | I **used to** go | ~~I use to go~~ |
| Negative | I **didn't use to** go | ~~didn't used to~~ |
| Question | **Did** you **use to** go? | ~~Did you used to go?~~ |
| Verb after | used to **go** | ~~used to going / went~~ |

> 🧠 **Memory hook:** **DID kills the D.** When *did/didn't* appears, *use*​**~~d~~** loses its **d**.
> 🇦🇿 *did/didn't* görəndə → **"use to"** (d-siz).

# 9. 🔔 Signal Words / Siqnal sözləri

These words love *used to / would*:

```
   when I was young     ·  as a child        ·  in those days
   back then            ·  in the past       ·  years ago
   every summer/winter  ·  every day (past)  ·  once / formerly
   before               ·  not anymore (the "now" contrast)
```

> 🇦🇿 *əvvəllər, keçmişdə, uşaqlıqda, o vaxtlar, hər yay, daha (artıq) yox.*

⚠️ Signal words like **every summer / when I was a child** are *required* to make **would** clear,
but only *helpful* (optional) for **used to**.

# 10. 🚫 Common Mistakes / Ən çox edilən səhvlər

```
 ❌ Did you used to smoke?
 ✅ Did you use to smoke?              (DID kills the D)

 ❌ I used to going to the gym.
 ✅ I used to go to the gym.           (V1 after used to)

 ❌ When I was a kid, I would have a bike.
 ✅ When I was a kid, I used to have a bike.   (have = state → no "would")

 ❌ I used to live here.   (...and I STILL live here)
 ✅ I live here.            (still true → Present Simple)

 ❌ I am used to wake up early.
 ✅ I used to wake up early.   (past habit)
    — OR, different meaning: I am used to waking up early. (I'm accustomed to it)  ← see §11
```

> 🇦🇿 **Azərbaycandan gələn səhv:** "-ardı/-ərdi" həm keçmiş, həm indiki vərdişi bildirə bilər,
> amma İngiliscə *used to* **yalnız keçmiş** üçündür. İndi də davam edirsə → **Present Simple**.

# 11. ⚖️ Comparison — the 3 "used to" forms / Üç "used to" forması

This is the **most confused area in the whole chapter.** Three look-alikes, three meanings:

```
 ┌────────────────────┬────────────────────────────┬───────────────────────────┐
 │  FORM              │  MEANING                   │  FOLLOWED BY              │
 ├────────────────────┼────────────────────────────┼───────────────────────────┤
 │ 1) used to + V1    │ past habit/state (finished)│  base verb (V1)           │
 │    I used to swim. │ keçmiş vərdiş → indi yox   │  used to SWIM             │
 ├────────────────────┼────────────────────────────┼───────────────────────────┤
 │ 2) be used to + …  │ I am ACCUSTOMED to it now  │  noun / V-ing             │
 │    I'm used to     │ artıq öyrəşmişəm (indi)    │  used to the COLD /       │
 │    the cold.       │                            │  used to WAKING up early  │
 ├────────────────────┼────────────────────────────┼───────────────────────────┤
 │ 3) get used to + … │ BECOMING accustomed        │  noun / V-ing             │
 │    I'm getting     │ öyrəşirəm (proses)        │  getting used to the JOB  │
 │    used to it.     │                            │                           │
 └────────────────────┴────────────────────────────┴───────────────────────────┘
```

🔑 **The deciding question:** *Is there a form of the verb **BE** (am/is/are/was) or **GET**
before "used to"?*

```
   NO  "be/get"  →  used to + V1     →  PAST HABIT       (I used to live…)
   YES "be"      →  be used to + -ing →  ACCUSTOMED NOW   (I'm used to living…)
   YES "get"     →  get used to + -ing →  BECOMING used   (I'm getting used to living…)
```

> 💡 **Test sentence:** *"When I moved to Norway, I **wasn't used to** the cold, but now I **am
> used to** it. I never **used to** wear a hat, but now I do."* — all three, one breath. Read it
> aloud 5 times. 🔁

### `used to` vs `would` vs Past Simple

```
 ┌─────────────┬────────────────────────────────┬──────────────────────┐
 │ FORM        │ USE                            │ STATE OK?            │
 ├─────────────┼────────────────────────────────┼──────────────────────┤
 │ Past Simple │ one finished action / a fact   │ ✅ (I was, I had)    │
 │ used to     │ repeated past habit + states   │ ✅ yes               │
 │ would       │ repeated past ACTIONS, nostalgic│ ❌ NO states        │
 └─────────────┴────────────────────────────────┴──────────────────────┘
```

# 12. 🇦🇿 Azerbaijani Translation Tips / Tərcümə məsləhətləri

You already own this grammar — Azerbaijani has a **dedicated past-habitual tense**: **-ardı /
-ərdi**. It maps onto *used to / would* almost perfectly. 🎉

```
   ENGLISH                          AZERBAIJANI ("-ardı/-ərdi")
   I used to go / I would go    →   gedərdim
   We used to play / would play →   oynayardıq
   She used to live there       →   orada yaşayardı
   They would come every Friday →   hər cümə gələrdilər
```

⚠️ **Three traps for Azerbaijani speakers:**

1. **Still-true habits.** Azerbaijani "-ardı" can feel general; English *used to* is **strictly
   past + finished**. If it's still true → use **Present Simple**.
   - *Mən çay içərəm* (I drink tea — still true) → **I drink tea**, NOT *I used to*.
2. **States need `used to`, not `would`.**
   - *Mənim itim **vardı*** → *I **used to have** a dog* ✅ (NOT *I would have*).
3. **"be used to" ≠ "used to".** Azerbaijani *öyrəşmişəm* (I'm accustomed) → **be used to +
   -ing**, a totally different structure. Don't mix it with the habit form.

> 🇦🇿 Qızıl qayda: **"-ardı/-ərdi" + keçmiş + artıq yox = used to / would.**
> **"öyrəşmişəm / öyrəşirəm" = be / get used to + -ing.**

# 13. 🔊 Pronunciation Notes / Tələffüz qeydləri

```
   used to        →  /ˈjuːs tə/   (sounds like "use-tuh" — the d is SILENT/blended!)
   didn't use to  →  /ˈdɪdənt juːs tə/
   would          →  /wʊd/  (short) ;  weak form in speech: /wəd/ → "we'd go"
   we would       →  often contracted:  we'd  /wiːd/
```

> 💡 **Huge tip:** *used to* is **almost never** pronounced /juːzd tuː/. The **d** disappears into
> the **t**, giving **/juːs tə/** ("yuus-tuh"). Say *"I yuus-tuh live here."* That's native. 🎯
> 🇦🇿 *used to* → "**yus-tu**" kimi səslənir, "d" demə.

Contractions you'll hear:
> *We'd go.* (= we would go) · *They'd visit.* (= they would visit)
> ⚠️ But *we'd* can also = *we **had** / we **would*** — context decides.

# 23. 🎤 Speaking Tips / Danışıq məsləhətləri

1. Use the **/juːs tə/** ("yus-tuh") pronunciation — it instantly sounds native.
2. Chain **would** for storytelling: *"Every summer we'd drive to the coast. We'd swim all day,
   and Dad would grill fish on the beach."* — set the scene once with **used to**, then flow with
   **would**. 🎯
3. Add the **"...but not anymore / now I…"** tail to show the contrast clearly.
4. For questions, default to **"Did you use to…?"** (not *would*).

---

# 24. 🖊️ Writing Tips / Yazı məsləhətləri

1. **Open with `used to`, continue with `would`.** Classic memoir rhythm:
   > *"My grandfather **used to** own a bakery. Every morning he **would** wake before dawn, he
   > **would** knead the dough by hand, and the whole street **would** smell of warm bread."*
2. Don't overuse **would** — readers may confuse it with the conditional. Anchor it with a
   time-phrase early.
3. For a **state** (be/have/own/like) always reach for **used to**, never *would*.

---

# 25. 🎧 Listening Tips / Dinləmə məsləhətləri

1. The **d is invisible**: you'll hear "**yus-tuh**," not "used-too." Train your ear for it.
2. **'d** contractions hide *would*: *"We'd visit"* = *we would visit*. Listen for **'d + base
   verb** = would; **'d + V3** = had.
   - *We'd **go*** → would · *We'd **gone*** → had
3. Storytellers (podcasts, interviews) pour out **would** when reminiscing — a signal they've
   shifted into "the old days."

---

# 26. 📖 Reading Tips / Oxu məsləhətləri

1. In memoirs & novels, a string of **would** = a remembered routine, not a future/conditional.
2. **used to** early in a paragraph quietly tells you: *"the following no longer happens."*
3. Watch the difference in print: **used to** (habit) vs **be used to** (accustomed). The little
   **am/is/are/was** before it changes everything.

---

# 27. 🗽 Native Speaker Usage / Doğma danışanlar necə işlədir

- Natives say **used to** dozens of times a day — it's not "bookish," it's everyday warmth.
- **"I used to..."** is how people open nostalgia, confessions, and "how I've changed" stories.
- **would** for habits feels slightly **literary / sentimental** — perfect for storytelling,
  toasts, eulogies, "the good old days."
- A super-common spoken frame: **"There used to be..."** for a place that changed.
- Natives almost never say *"used to"* for something still true — that instantly signals "this is
  over."

# 28. ❓ FAQ / Tez-tez verilən suallar

**Q1. Is `used to` a tense?**
No. It's a **special structure** about the past. That's why it sits in Part 4, not in the 12-tense
grid.

**Q2. What's the difference between `used to` and `would`?**
`used to` = habits **and** states; works alone. `would` = repeated **actions only**, needs a past
time-frame, and can't take states (be/have/like). 👉 §11.

**Q3. Is it "Did you use to" or "Did you used to"?**
✅ **use to** (no -d). *Did/didn't* already carries the past, so the verb returns to base. 🧠 "DID
kills the D."

**Q4. `used to` vs `be used to`?**
Totally different! *used to + V1* = past habit. *be used to + -ing/noun* = accustomed **now**.
*"I used to drive"* (past) ≠ *"I'm used to driving"* (no problem for me now). 👉 §11.

**Q5. Can I say "I used to live in Baku" if I still live there?**
No. That implies you **left**. Still there → *"I live in Baku."*

**Q6. How do I make `used to` negative?**
*didn't use to* (most common) or *never used to*. ✅ *I didn't use to like it.*

**Q7. How does this map to Azerbaijani?**
Beautifully → **-ardı/-ərdi** (*gedərdim, yaşayardı*). But beware: if it's still true, switch to
Present Simple; and *öyrəşmişəm* = **be used to**, not *used to*. 👉 §12.

```
 └───────────────────────────────────────────────────────────────────────┘
```

> 🧠 **One-line memory:** *"used to = the way things **were** and aren't now;
> would = the things we **would** do, again and again, back then."*

## 🇬🇧 The House on Olive Street

Grandpa's house **used to** stand at the end of Olive Street, where the road turned to dust.

When I was small, every summer **would** begin the same way. We **would** arrive after dark, and
Grandma **would** be waiting on the porch with a lamp. There **used to be** a huge fig tree in the
yard, and my cousins and I **would** climb it until someone called us for dinner.

Grandpa **used to** be a carpenter. He **didn't use to** say much, but his hands were always busy.
Every morning he **would** sit by the window, sharpening his tools, and the whole house **would**
smell of wood and coffee. I **used to** think he was the strongest man in the world.

In the evenings, the neighbours **would** come over. They **would** drink tea, play cards, and
argue about football. We children **used to** hide under the table and listen. Nobody **used to**
lock their doors back then — people just walked in.

I **didn't use to** understand why my mother cried when we left each September. I understand now.

Last week I drove down Olive Street. The fig tree is gone. The house is gone. There's a car park
where the porch **used to** be. For a moment I just sat there, the engine running.

Some things you don't get used to. You just learn to carry them.

# 🧩 EXPANSION PACK — *used to / would* (E1–E12)
### Deeper, trickier, more native. Read slowly. ☕

> 🎯 You already have the basics (§1–§29). This pack handles the **edges** — the places where
> learners (especially Azerbaijani speakers) slip. Each item is short, with ✅/❌ and 🇦🇿 notes.

---

## E1 · `would` ≠ a one-time action ⚠️
**`would` only works for things that happened AGAIN and AGAIN.** A single, finished event needs
**Past Simple** (or *used to* never works here either).

```
 ❌ Yesterday, I would call my friend.        (one time → wrong)
 ✅ Yesterday, I called my friend.            (Past Simple)

 ✅ When I was young, I would call her every night.   (repeated → ok)
```
> 🧠 Test: can you add **"every day / always / again and again"**? If **no**, don't use *would*.
> 🇦🇿 *would* yalnız **təkrarlanan** hərəkət üçündür. Bir dəfəlik iş → Past Simple.

---

## E2 · The `wouldn't` = "refused" trap 🚪
In the **past**, **wouldn't** often means **refused to** — not "didn't use to."

```
 The car wouldn't start this morning.     = it REFUSED to start (one event)
 As a child, I wouldn't eat spinach.       = I REFUSED / habit of refusing
```
So for a simple "didn't have the habit," prefer **didn't use to** to avoid confusion:
```
 ✅ I didn't use to like spinach.   (clear: no habit)
 ⚠️ I wouldn't eat spinach.         (sounds like stubborn refusal)
```
> 🇦🇿 *wouldn't* çox vaxt **"imtina etdi / inad etdi"** mənası verir. Sадə "vərdişim yox idi"
> üçün **didn't use to** işlət.

---

## E3 · `would` (habit) vs `would have` (conditional) — totally different! 🔀
Don't confuse the **habit** *would* with the **conditional/unreal** *would (have)*.

```
 HABIT (real past):    Every summer we would swim in the lake.   (= used to)
 CONDITIONAL (unreal): I would swim, but the water is too cold.  (if-situation)
 UNREAL PAST:          I would have swum, but it rained.         (3rd conditional)
```
🔑 Habit *would* = **real**, repeated, has a past time-frame. Conditional *would* = **imagined**,
tied to an *if*. Chapter 16 (conditionals) covers the second kind. 🇦🇿 Birincisi **real keçmiş
vərdiş**, ikincisi **şərti/xəyali**.

## E4 · The 3-way drill, expanded — `used to` / `be used to` / `get used to` 🎯
More contrast pairs to burn it in:

```
 ┌──────────────────────────────┬─────────────────────────────────────┐
 │ used to + V1 (PAST HABIT)     │ be/get used to + V-ing (ACCUSTOMED) │
 ├──────────────────────────────┼─────────────────────────────────────┤
 │ I used to drive to work.      │ I'm used to driving in traffic.     │
 │ (I don't anymore)             │ (no problem for me now)             │
 │ She used to live alone.       │ She's getting used to living alone. │
 │ We used to work nights.       │ We're used to working nights.       │
 └──────────────────────────────┴─────────────────────────────────────┘
```
🔑 **Decider:** Is there **am/is/are/was/get** right before *used to*?
**Yes → accustomed (+ -ing)** · **No → past habit (+ V1)**.

> 💡 Killer minimal pair, say it aloud:
> *"I **used to** live in a city."* (past, finished) ↔ *"I**'m used to** living in a city."*
> (I cope fine now). One little **'m** flips the meaning 180°.

## E5 · Adverb placement (always / never / often) 📍
With **would / used to**, frequency adverbs sit **between** the helper and the verb:

```
 ✅ He would ALWAYS arrive late.
 ✅ We used to OFTEN go camping.   (also ok: We often used to go camping.)
 ✅ I would NEVER eat breakfast back then.
 ❌ He always would arrive late.   (clumsy)
```
> 🇦🇿 *always/never/often* → köməkçi ilə əsas felin **arasında**: *would always go*.

---

## E6 · `used to` carries a silent "...but not now" 🤫
*used to* doesn't just describe the past — it **implies the present is different**. That contrast
is doing real work in conversation:

```
 "I used to be shy."        → (and now I'm NOT shy — I've changed)
 "She used to call me."     → (but she doesn't anymore — maybe something's wrong)
 "This used to be easier."  → (it's harder now — a quiet complaint)
```
> 💡 Natives use *used to* to **hint at change** without stating it. The listener fills in "...but
> not now." 🇦🇿 *used to* gizli "**...amma indi yox**" mesajı daşıyır.

## E7 · Tag questions & short answers 🏷️
```
 You used to smoke, didn't you?      → Yes, I did. / No, I didn't.
 She didn't use to like him, did she? → No, she didn't.
 There used to be a shop here, didn't there?
```
🔑 The tag uses **did/didn't** (because *used to* behaves like a past verb), **not** *usedn't*
(very old/rare). 🇦🇿 Tag sualı **did/didn't** ilə qurulur.

## E8 · The "would" storytelling chain (a native writing move) ✍️📜
This is how good writers paint a **remembered routine**: open with **used to**, then pour out a
chain of **would**.

```
 "My grandmother USED TO run a tiny shop. Every morning she WOULD unlock the door at six,
  she WOULD sweep the step, she WOULD line up the jars, and by seven the first customers
  WOULD already be knocking."
```
- **used to** = sets the stage (the general fact)
- **would, would, would** = the repeated little actions inside that stage
> 💡 Use this in IELTS Writing/Speaking Part 2 ("Describe a person/place from your past") for a
> Band-7+ effect. 🇦🇿 Əvvəl **used to** ilə səhnə qur, sonra **would** zənciri ilə danış.

## E9 · `didn't use to` vs `never used to` — same idea, different flavour
```
 I didn't use to like jazz.   (neutral, standard negative)
 I never used to like jazz.    (slightly stronger / more emphatic, very common in speech)
```
Both are correct. **never used to** is extremely common in casual British/American speech.
⚠️ Don't double it: ❌ *I didn't never use to* (double negative).
> 🇦🇿 İkisi də düzgündür; **never used to** danışıqda daha çox işlənir.

## E10 · 🗺️ THE MASTER DECISION MAP — past habits & "used to" family

```
 ┌───────────────────────────────────────────────────────────────────────────┐
 │  START: I want to talk about the past / a habit / being accustomed.        │
 └───────────────────────────────────────────────────────────────────────────┘
        │
        ▼
  Is there "be" or "get" (am/is/are/was/get) right before "used to"?
        │
        ├── YES → it's about being ACCUSTOMED (now/process), NOT a past habit
        │         ├── be used to + -ing/noun   → "I'm used to the cold."
        │         └── get used to + -ing/noun  → "I'm getting used to it."
        │
        └── NO → it's a PAST HABIT / STATE
                 │
                 ▼
            Is it still true now?
                 ├── YES → use PRESENT SIMPLE  ("I live in Baku.")  ❌ not used to
                 └── NO  → it's finished → continue
                          │
                          ▼
                    Is it a STATE? (be / have / like / know / own / want)
                          ├── YES → ONLY "used to"      ("I used to have a dog.")
                          └── NO (an action) → 
                                   ├── general fact?      → used to  ("I used to swim.")
                                   ├── repeated + scene?   → would    ("I'd swim every day.")
                                   └── ONE time only?      → PAST SIMPLE ("I swam yesterday.")
```

🇦🇿 **Qısa yol:**
```
   be/get + used to + -ing  →  öyrəşmişəm / öyrəşirəm   (indi)
   used to + V1 (state)     →  vardı / idi              (keçmiş, bitmiş)
   used to / would + V1     →  -ardı/-ərdi              (təkrar vərdiş)
   bir dəfə                  →  Past Simple              (-dı/-di)
   hələ də doğrudur          →  Present Simple
```

## E12 · 🇦🇿 Azerbaijani-interference mistakes (deep cut)
```
 ❌ I am used to play football when I was a child.
 ✅ I used to play football when I was a child.
    → "öyrəşmişəm" (am used to) ilə "oynayardım" (used to) qarışdırılır.

 ❌ I used to studying English for 3 years. (meaning: I've studied…)
 ✅ I have studied / have been studying English for 3 years.
    → "used to" davam edən müddət üçün DEYİL; bu Present Perfect-dir.

 ❌ Before, I would have a bicycle.
 ✅ Before, I used to have a bicycle.   (have = state → no "would")

 ❌ I used to go there. (...and I STILL go there)
 ✅ I (still) go there.   (hələ də doğrudur → Present Simple)

 ❌ When I was a student, I would be lazy.
 ✅ When I was a student, I used to be lazy.   (be = state)
```
> 🧠 The 4 deep traps for AZ speakers: **(1)** *am used to* ≠ *used to*; **(2)** *used to* is not
> for ongoing duration (that's Present Perfect); **(3)** states reject *would*; **(4)** still-true
> → Present Simple.

# 📖 MINI STORY #2 — *"A New City"*
### (used to / would AND be/get used to — all in one story) 🌧️🏙️

> 🎯 Read the 🇬🇧 first. **Bold** = the chapter's forms. Watch two kinds:
> 🟦 *used to / would* = the **old life** (past, finished) · 🟩 *be/get used to* = **adapting now**.
> Then read the 🇦🇿 and do the questions.

## 🇬🇧 A New City

Back home, the sun **used to** wake me before my alarm.

In my village, every morning **would** start the same way. My mother **would** make tea on the old
stove, the rooster next door **would** crow, and I **would** walk to the bakery while the streets
were still empty. I **used to** know every face I passed. There **used to be** a bench under the
walnut tree where the old men **would** sit and argue about everything and nothing.

Then I moved to London.

At first, I hated it. I **wasn't used to** the rain, the grey sky, the crowds that never looked at
you. Back home, neighbours **would** stop and talk for an hour; here, people just hurried past. I
**wasn't used to** eating dinner alone, and I **wasn't used to** the silence of my little flat.

"You'll **get used to** it," my flatmate kept saying. I didn't believe her.

But slowly, something changed. I **got used to** the trains. I **got used to** the strange,
beautiful mix of languages on the bus. I even **got used to** the rain — I bought a good coat and
stopped complaining.

Now, two years later, I **am used to** this city. I **am used to** the noise, the speed, the
endless grey mornings. I have my own little routine: I buy the same coffee, I take the same seat
by the window.

And yet — some mornings, when the light is soft, I still remember the rooster, the walnut tree,
the bench. I **used to** think I would never belong here.

I was wrong. You don't replace the old home. You just learn to carry two.

haqqında mübahisə edərdilər.

Sonra Londona köçdüm.

Əvvəlcə ondan zəhləm gedirdi. Yağışa, boz səmaya, sənə heç vaxt baxmayan izdihama öyrəşməmişdim.
Doğma yurdumda qonşular dayanıb bir saat söhbət edərdi; burada isə insanlar sadəcə tələsib keçirdi.
Tək şam yeməyə öyrəşməmişdim, kiçik mənzilimin sükutuna da öyrəşməmişdim.

"Öyrəşəcəksən," deyə otaq yoldaşım təkrar edirdi. Ona inanmırdım.

Amma yavaş-yavaş nəsə dəyişdi. Qatarlara öyrəşdim. Avtobusdakı qəribə, gözəl dil qarışığına
öyrəşdim. Hətta yağışa da öyrəşdim — yaxşı bir palto aldım və şikayət etməyi dayandırdım.

İndi, iki il sonra, bu şəhərə öyrəşmişəm. Səs-küyə, sürətə, sonsuz boz səhərlərə öyrəşmişəm. Öz

ki, heç vaxt buraya aid olmayacağam.

Səhv edirdim. Köhnə yurdu əvəz etmirsən. Sadəcə iki yurdu daşımağı öyrənirsən.

## 🔑 Story key — two colours, two meanings

🟦 **OLD LIFE — `used to` / `would`** (past, finished):
```
 the sun used to wake me        → günəş məni oyadardı
 every morning would start...    → hər səhər başlayardı
 my mother would make tea        → anam çay dəmləyərdi
 I would walk to the bakery      → çörəkxanaya gedərdim
 I used to know every face       → hər üzü tanıyardım   (know = state → used to!)
 there used to be a bench         → bir skamya var idi
 the old men would sit and argue  → qocalar oturub mübahisə edərdi
```

🟩 **ADAPTING — `be / get used to + -ing/noun`** (accustomed):
```
 I wasn't used to the rain        → yağışa öyrəşməmişdim     (be used to + noun)
 wasn't used to eating alone       → tək yeməyə öyrəşməmişdim (be used to + V-ing)
 You'll get used to it             → öyrəşəcəksən             (get used to)
 I got used to the trains          → qatarlara öyrəşdim       (get used to + noun)
 I am used to this city now         → bu şəhərə öyrəşmişəm    (be used to)
```

> 🧠 See the **whole chapter** in one story: the 🟦 *old habits* are **finished** (used to /
> would), while the 🟩 *adapting* is about **getting comfortable now** (be/get used to + -ing).
> Notice **"I used to know"** — *know* is a STATE, so it takes *used to*, never *would*. ✅
