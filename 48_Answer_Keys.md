# 2. ✅ HOW TO CHECK YOURSELF HONESTLY / Özünü düzgün yoxla

The answer key only helps if you use it the right way. 🇦🇿 *Cavab açarı yalnız düzgün istifadə olunanda kömək edir.*

```
   1. COVER first.   Do the whole exercise before you look. No peeking.
                     🇦🇿 Əvvəlcə hamısını həll et, sonra bax.

   2. MARK, don't move on.   ✔ correct · ✘ wrong · ❓ "guessed it right".
                     🇦🇿 Təxminən tapdığını da "❓" işarələ — onu da təkrar et.

   3. ASK "WHY", not "what".   For every ✘, write the RULE you broke (see §5).
                     🇦🇿 Səhvin "nə" olduğunu yox, "niyə" olduğunu yaz.

   4. REDO after a delay.   Redo wrong items the NEXT day, from scratch.
                     🇦🇿 Səhvləri ertəsi gün sıfırdan yenidən həll et.

   5. SAY it aloud.   Read the corrected sentence out loud 3×.
                     🇦🇿 Düzəldilmiş cümləni 3 dəfə ucadan oxu.
```

> ⚠️ A right answer for the wrong reason is still a problem. 🇦🇿 Yanlış səbəbə görə tapılmış düzgün
> cavab yenə də problemdir — onu "❓" kimi qeyd et və qaydasını yoxla.

# 5. ⚠️ TOP MISTAKES — master checklist / Ən çox təkrarlanan səhvlər

Before checking any answer key, scan your sentences for these book-wide traps:

```
 PRESENT
  □ 3rd person singular -s:  he/she/it works (not "work")
  □ Stative verbs: no -ing  →  I know (not "I am knowing")
  □ Present Perfect vs Past Simple:  finished time → Past Simple

 PAST
  □ Past Simple after "did": did + V1  →  Did you go? (not "Did you went?")
  □ was/were:  I/he/she/it WAS · you/we/they WERE
  □ Past Perfect for the EARLIER past action: had + V3

 FUTURE
  □ will + V1 (bare):  will go (not "will to go")
  □ Time clauses use PRESENT, not will:  When I arrive (not "when I will arrive")
  □ going to for plans/evidence · will for decisions/predictions

 ADVANCED
  □ Conditionals: 1st → if + present, will · 2nd → if + past, would · 3rd → if + had V3, would have V3
  □ Reported speech BACKSHIFT: "I am" → he said he WAS
  □ Passive: be + V3  →  It was written (correct V3, not V2)

 GENERAL
  □ a/an/the (articles don't exist in Azerbaijani — double-check!)
  □ Word order S–V–O (English), not S–O–V
  □ V2 vs V3:  wrote (V2) ≠ written (V3)
```

> 🇦🇿 Bunlar bütün kitab boyu xəbərdarlıq etdiyimiz **əsas tələlərdir**. Cavaba baxmazdan əvvəl
> cümlələrini bu siyahı ilə yoxla — çox vaxt səhvi özün tapacaqsan.
